# <a name="dashboard-not-loading"></a>Tableau de bord non chargé

**Symptôme :**

Le tableau de bord peut être bloqué par une toupie de chargement, ou il peut afficher un message d'erreur générique : « Quelque chose a mal tourné ». Pour plus de détails, consultez les détails techniques ».

**Guide de dépannage :**

Veuillez voir [le tableau de bord UCI qui ne se charge pas après la version 2003.5](https://support.microsoft.com/help/4558635/uci-dashboard-not-loading-after-the-2003-5-release).

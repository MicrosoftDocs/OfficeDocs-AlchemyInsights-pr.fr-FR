# <a name="sharepoint-alert-notifications-not-delivered"></a>Notifications d’alerte SharePoint non remises 

<p style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;"><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Déterminez si <strong>toutes les alertes</strong> ne sont pas remises ou si une <strong>alerte individuelle</strong> d’un fichier ou d’une bibliothèque spécifique n’est pas remise.</span></p> <ol> <li><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Si <strong style="mso-bidi-font-weight: normal;">toutes les alertes</strong> provenant de plusieurs fichiers ou bibliothèques ne sont pas remises, visitez le <a href="https://admin.microsoft.com/AdminPortal/Home#/servicehealth">tableau de bord état des services</a> pour vérifier les avis/incidents susceptibles de se produire avec SharePoint ou Exchange. Le problème peut être lié à la capacité d’alerte ou aux retards d’alertes de SharePoint par le biais d’Exchange. Il est également important de noter si d’autres e-mails sont remis et, si ce n’est pas le cas, le problème est probablement lié à Exchange. <br /><br /></span></li> <li style="line-height: normal; font-size: 11pt; font-style: normal; font-weight: 400;"><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Si une</span> <strong><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">alerte individuelle</span></strong> <span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">à partir d’un fichier ou d’une bibliothèque spécifique n’est pas remis, vous pouvez essayer de supprimer et de recréer. Consultez la rubrique</span> <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> <a href="https://support.office.com/en-us/article/manage-view-or-delete-sharepoint-alerts-99dfb19c-9a90-4a8c-aba1-aa8c8afb0de2#ID0EAADAAA=Online">Manage, View, or delete SharePoint Alerts</a> pour recréer l’alerte.</span></li> </ol> <p style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;"><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Pour plus d’informations sur la configuration des alertes, consultez <span style="color: #333333;"> <a href="https://support.office.com/en-us/article/create-an-alert-to-get-notified-when-a-file-or-folder-changes-in-sharepoint-e5a79e7b-a146-46da-a9ef-d65409ba8918">la rubrique créer une alerte pour être averti lorsqu’un fichier ou un dossier est modifié dans SharePoint</a>.</span></span></p> <p style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;"><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Pour plus d’informations sur la résolution des alertes, consultez la rubrique <a href="https://support.office.com/en-us/article/users-don-t-receive-sharepoint-online-alert-notifications-14fc22dd-e873-482c-844d-f67ad41313f1">les utilisateurs ne reçoivent pas les&rsquo;notifications d’alerte SharePoint Online</a>.</span></p> <p>&nbsp;</p>

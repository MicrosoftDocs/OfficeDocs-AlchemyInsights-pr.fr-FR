# <a name="change-the-font-size-in-a-message"></a>Modifier la taille de la police dans un message

1. Si vous avez déjà tapé le message, sélectionnez le texte.
2. Sélectionnez l’icône de  **mise en forme** sous le message (il s’agit de la première icône).
3. Sélectionnez l’icône de  **taille de police**  au-dessus du message (il s’agit de la septième icône).
4. Sélectionnez une taille.

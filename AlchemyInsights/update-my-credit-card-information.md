# <a name="update-my-credit-card-information"></a>Mettre à jour mes informations de carte bancaire

Vous pouvez ajouter ou supprimer un mode de paiement dans le Centre d’administration **Microsoft 365**. Si votre mode de paiement n’est associé à aucun abonnement, vous pouvez le supprimer immédiatement. S’il est associé à un abonnement, consultez [Modifier un mode de paiement](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/add-update-or-remove-credit-card-or-bank-account?view=o365-worldwide#change-a-payment-method).

1. Accédez à **Facturation > Factures et paiements > [Modes de paiement](https://go.microsoft.com/fwlink/p/?linkid=2018806)**.

2. Pointez sur la carte ou le compte bancaire, puis sélectionnez l’icône de suppression.

3. Au bas du volet **Supprimer le mode de paiement**, cliquez sur **Supprimer**.

Pour plus d’informations, veuillez consulter la page [Ajouter, mettre à jour ou supprimer vos modes de paiement](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/add-update-or-remove-credit-card-or-bank-account?view=o365-worldwide#update-an-existing-payment-method).

Pour résoudre ce problème, consultez [Résolution des problèmes des modes de paiement](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/add-update-or-remove-credit-card-or-bank-account?view=o365-worldwide#troubleshoot-payment-methods).

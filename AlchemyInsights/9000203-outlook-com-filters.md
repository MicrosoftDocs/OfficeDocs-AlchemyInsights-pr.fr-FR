# <a name="filtering-in-outlookcom"></a>Filtrage dans Outlook.com

Vous pouvez filtrer vos messages électroniques de plusieurs façons dans Outlook.com:

1. [Modifier le mode de tri de votre courrier électronique](https://support.office.com/article/e650ae23-b558-4fbf-bdd1-73268f6852b7)
2. [Bloquer une adresse de messagerie ou un domaine de messagerie](https://support.office.com/article/afba1c94-77bb-4f50-8b85-057cf52f4d5e)
3. [Utilisation de la recherche avancée de requête pour affiner les résultats](https://support.office.com/article/88108edf-028e-4306-b87e-7400bbb40aa7)

# <a name="update-dns-records-to-keep-your-website-with-your-current-hosting-provider"></a>Mettre à jour les enregistrements DNS pour conserver votre site web chez votre fournisseur d'hébergement actuel

Dans le cadre du processus de déplacement de votre site web public Microsoft 365 vers un autre fournisseur d’hébergement, vous devrez probablement apporter des modifications à vos enregistrements de domaine.
  
1. Dans le centre d’administration, sélectionnez **le** nom de domaine à modifier dans le programme d’installation \>  des domaines.

2. Entrez les informations qui s’alignent sur votre site web avec votre nouveau fournisseur d’hébergement.

Pour plus d’informations, voir Mettre à jour les enregistrements DNS pour conserver votre [site web auprès de votre fournisseur d’hébergement actuel.](https://docs.microsoft.com/microsoft-365/admin/dns/update-dns-records-to-retain-current-hosting-provider?view=o365-worldwide)
  
Pour en savoir plus sur l’interruption de site public, voir Sites web publics [SharePoint Online à arrêter.](https://support.office.com/article/sharepoint-online-public-websites-to-be-discontinued-e86bfd2f-5c7d-446f-a430-7cfcc0130916)

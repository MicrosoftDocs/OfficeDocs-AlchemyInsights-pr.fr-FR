# <a name="update-dns-records-to-keep-your-website-with-your-current-hosting-provider"></a>Mettre à jour les enregistrements DNS pour conserver votre site web chez votre fournisseur d'hébergement actuel

Dans le cadre du processus de déplacement de votre site de Web Office 365 Public vers un autre fournisseur d’hébergement, vous devrez probablement apporter des modifications aux enregistrements de votre domaine.
  
1. Dans le centre d’administration, accédez à **configuration** \> **domaines** , choisissez le nom de domaine que vous souhaitez modifier. 
    
2. Entrez les informations qui s’aligne sur votre site Web avec votre fournisseur d’hébergement.
    
Pour plus d’informations, voir [enregistrements DNS de mise à jour pour conserver votre site Web de votre fournisseur d’hébergement actuel](https://support.office.com/en-us/article/update-dns-records-to-keep-your-website-with-your-current-hosting-provider-2c4cf347-b897-45c1-a71f-210bdc8f1061). 
  
Pour plus d’informations sur l’arrêt du site public, voir [SharePoint Online les sites Web publics pour être supprimées](https://support.office.com/en-us/article/sharepoint-online-public-websites-to-be-discontinued-e86bfd2f-5c7d-446f-a430-7cfcc0130916?ui=en-US&amp;rs=en-US&amp;ad=US).

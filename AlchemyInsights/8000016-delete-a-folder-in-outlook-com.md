# <a name="delete-a-folder"></a>Supprimer un dossier

1. Dans le volet gauche, cliquez sur le dossier ou le sous-dossier que vous souhaitez supprimer, sélectionnez **Supprimer**. 
    
2. Cliquez sur **OK** pour confirmer. 
    
Pour plus d’informations, voir [suppression d’un dossier dans Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873134).

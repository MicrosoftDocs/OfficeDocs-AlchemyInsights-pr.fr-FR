# <a name="yammer---possible-service-issue"></a>Yammer : problème de service possible

Il peut y avoir un problème temporaire : les utilisateurs peuvent être confronté à des erreurs lors de l’accès à Yammer. Consultez le [tableau de bord d’État du service](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) pour voir si cela affecte votre réseau.

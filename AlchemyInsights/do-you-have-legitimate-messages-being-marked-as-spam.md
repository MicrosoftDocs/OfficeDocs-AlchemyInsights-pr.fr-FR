# <a name="do-you-have-legitimate-messages-being-marked-as-spam"></a>Les messages légitimes sont-ils marqués comme courrier indésirable?


Essayez le diagnostic ci-dessus ou essayez ce [Guide pas à pas pour la résolution des messages correctement marqués comme courrier](https://docs.microsoft.com/office365/securitycompliance/prevent-email-from-being-marked-as-spam-0)indésirable.
  
La meilleure façon de signaler ces types de problèmes est de:
  
1. [Activer le complément de création de rapports](https://docs.microsoft.com/office365/securitycompliance/enable-the-report-message-add-in) pour l'utilisateur. 
    
2. Demandez à l'utilisateur de [signaler le message comme légitime](https://support.office.com/article/use-the-report-message-add-in-b5caa9f1-cdf3-4443-af8c-ff724ea719d2?ui=en-US&amp;rs=en-US&amp;ad=US).
    
Si vous souhaitez ouvrir un ticket de support, veuillez joindre les en-têtes de messages complets d'au moins **un message d'origine** ci-dessous.

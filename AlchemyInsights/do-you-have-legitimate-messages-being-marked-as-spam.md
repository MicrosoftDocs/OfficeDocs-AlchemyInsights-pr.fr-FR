# <a name="do-you-have-legitimate-messages-being-marked-as-spam"></a>Avez-vous besoin de messages légitimes est marqués comme courrier indésirable ?


Essayez le diagnostic ci-dessus ou ce [guide étape par étape pour résoudre les problèmes des messages est marqués comme courrier indésirable](https://docs.microsoft.com/office365/securitycompliance/prevent-email-from-being-marked-as-spam-0).
  
La meilleure façon pour signaler ces types de problèmes consiste à :
  
1. [Activer le complément reporting](https://docs.microsoft.com/office365/securitycompliance/enable-the-report-message-add-in) pour l’utilisateur. 
    
2. Ont l' [état le message comme légitime](https://support.office.com/article/use-the-report-message-add-in-b5caa9f1-cdf3-4443-af8c-ff724ea719d2?ui=en-US&amp;rs=en-US&amp;ad=US)de l’utilisateur.
    
Si vous souhaitez ouvrir un ticket de support, attachez les en-têtes de messagerie complète au moins **un message d’origine entier** ci-dessous.

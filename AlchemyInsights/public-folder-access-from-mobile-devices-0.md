# <a name="public-folder-access-from-mobile-devices"></a>Accès aux dossiers publics depuis des appareils mobiles

Dossiers publics ne sont pas accessibles depuis des appareils mobiles.
  
Si vous souhaitez disposent de fonctionnalités similaires aux dossiers publics et d’y accéder depuis des appareils mobiles, voir [en savoir plus sur les groupes d’Office 365](https://support.office.com/en-us/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) pour une meilleure solution.

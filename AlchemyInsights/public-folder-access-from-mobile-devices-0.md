# <a name="public-folder-access-from-mobile-devices"></a>Accès aux dossiers publics à partir d’appareils mobiles

Les dossiers publics ne sont pas accessibles à partir des appareils mobiles.
  
Si vous souhaitez utiliser des fonctionnalités similaires aux dossiers publics et y accéder à partir d’appareils mobiles, consultez la rubrique [en savoir plus sur les groupes Office 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) pour une meilleure solution.

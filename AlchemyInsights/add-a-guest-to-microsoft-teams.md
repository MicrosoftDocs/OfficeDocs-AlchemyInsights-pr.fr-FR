# <a name="add-a-guest-to-microsoft-teams"></a>Ajouter un invité à Microsoft teams

Parfois, vous voudrez collaborer avec des personnes qui ne sont pas dans l’organisation de votre équipe. Vous pouvez souvent le faire par courrier électronique, mais si vous souhaitez collaborer avec eux dans Teams, vous pouvez les ajouter en tant qu’invités.

Pour plus d’informations, consultez [la rubrique ajouter des invités à une équipe dans teams](https://support.office.com/article/add-guests-to-a-team-in-teams-fccb4fa6-f864-4508-bdde-256e7384a14f#ID0EAABAAA=Desktop).

**Remarque :** Vous devez être propriétaire de l’équipe pour pouvoir ajouter un invité dans Teams (également voir [définir les autorisations invitées](https://support.office.com/article/set-guest-permissions-for-channels-in-teams-4756c468-2746-4bfd-a582-736d55fcc169)).

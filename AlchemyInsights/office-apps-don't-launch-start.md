# <a name="microsoft-365-apps-crash-stop-responding-or-dont-launch"></a>Blocage des applications Microsoft 365, blocage ou absence de lancement

Pour résoudre les problèmes d’applications Microsoft 365, essayez les options suivantes :

- [Examiner les problèmes possibles avec les compléments](https://support.office.com/article/powerpoint-isn-t-responding-hangs-or-freezes-652ede6e-e3d2-449a-a07f-8c800dfb948d#bkmk_addins) installés pour Office
- Vérifier si le [logiciel antivirus est en conflit](https://support.office.com/article/powerpoint-isn-t-responding-hangs-or-freezes-652ede6e-e3d2-449a-a07f-8c800dfb948d?ocmsassetID#bkmk_conflict) avec les applications Microsoft 365
- Vérifier ou [installer les dernières mises à jour pour Office](https://support.office.com/article/update-office-and-your-computer-with-microsoft-update-2ab296f3-7f03-43a2-8e50-46de917611c5)
- [Réparer Office](https://support.office.com/article/repair-an-office-application-7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b) à l’aide du panneau de configuration
- [Désinstaller](https://support.office.com/article/uninstall-office-from-a-pc-9dd49b83-264a-477a-8fcc-2fdf5dbf61d8) et réinstaller Office

Si vous souhaitez en savoir plus, consultez les articles :

- [PowerPoint ne répond pas, se bloque ou se bloque](https://support.office.com/article/powerpoint-isn-t-responding-hangs-or-freezes-652ede6e-e3d2-449a-a07f-8c800dfb948d)

- [Renforcez la protection de vos fichiers en cas de panne](https://support.office.com/article/help-protect-your-files-in-case-of-a-crash-551c29b1-6a4b-4415-a3ff-a80415b92f99)

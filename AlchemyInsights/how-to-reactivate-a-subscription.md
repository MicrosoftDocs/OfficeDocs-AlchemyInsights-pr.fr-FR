# <a name="how-to-reactivate-a-subscription"></a>Comment réactiver un abonnement

Si votre abonnement a expiré, a été annulé ou a été désactivé par Microsoft, vous pouvez le réactiver via le Centre d’Administration Microsoft 365.
  
1. Dans le Centre d’administration, choisissez la page **Facturation** \> [Abonnements](https://go.microsoft.com/fwlink/p/?linkid=842054).

2. Sélectionnez l’abonnement à réactiver.

3. Sélectionnez le bouton **Réactiver** à droite.

4. Vous êtes invité à mettre à jour les détails de paiement. Lorsque toutes les informations sont correctes, sélectionnez **Envoyer**.

5. Une confirmation s’affiche au bas de l’écran. Sélectionnez **Fermer** pour revenir à la page **Abonnements**.

6. Si l’abonnement n’apparaît pas actif après quelques minutes, actualisez la page.

Pour plus d’informations, voir: [Réactiver ou renouveler un abonnement Office 365 Entreprise](https://docs.microsoft.com/fr-FR/office365/admin/subscriptions-and-billing/renew-your-subscription).

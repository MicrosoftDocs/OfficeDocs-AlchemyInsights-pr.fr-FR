# <a name="creating-email-signatures"></a>Création de signatures électroniques

Ces articles devraient vous aider à créer les signatures électroniques que vous recherchez.
  
- Je veux créer une signature électronique à l’aide d’Outlook sur un ordinateur Windows : [créer et ajouter une signature aux messages](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx)
    
- Je veux créer une signature électronique à l’aide d’Outlook sur le web : [créer et ajouter une signature électronique dans Outlook sur le web](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx)
    
- Je veux créer une signature électronique à l’aide de Outlook.com : [créer et ajouter une signature électronique dans Outlook.com](https://support.office.com/article/776d9006-abdf-444e-b5b7-a61821dff034.aspx)
    
- Je veux créer une signature électronique global pour tous les messages électroniques envoyés à partir de ma société : [créer des signatures de l’organisation et des exclusions dans Office 365](https://support.office.com/article/2d75860f-c527-4352-a7f6-73eba54c0c72.aspx)
    
 **Conseil :** Échelle de l’organisation des signatures sont appelées « responsabilité ».

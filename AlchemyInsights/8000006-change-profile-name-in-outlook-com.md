# <a name="change-your-profile-name"></a>Modifier le nom de votre profil

1. Accédez à [votre Microsoft les informations de compte](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. Sélectionnez **Modifier le nom**. 
    
3. Entrez le nom et le nom de famille, puis sélectionnez **Enregistrer**. 
    
Patientez jusqu'à 24 heures pour que les modifications de votre nom de profil à prendre effet.
  
Pour plus d’informations, voir [modifier votre nom dans Outlook.com ou les photos de profil](https://go.microsoft.com/fwlink/?linkid=873110).

# <a name="emerging-issue"></a>Problème émergeant

Plusieurs clients se plaignent d’un problème semblable à celui que vous signalez. Nous nous penchons sur ce problème.

Consultez le [tableau de bord d’État du service](https://admin.microsoft.com/adminportal/home#/servicehealth) pour rester à jour sur tous les services ayant un impact sur les incidents.

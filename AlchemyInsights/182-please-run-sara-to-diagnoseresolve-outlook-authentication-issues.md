# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Utilisez SaRA pour diagnostiquer et résoudre les problèmes d’authentification Outlook

Nous vous recommandons de que vous utilisez la [demande mon mot de passe Outlook](https://aka.ms/SaRA-OutlookPwdPrompt) diagnostic sur la machine affectée pour résoudre les problèmes où Outlook en permanence vous invite à fournir un mot de passe. Cette [SaRA](https://diagnostics.office.com/#/) diagnostic effectue des vérifications automatiques et renvoie les solutions possibles, vous pouvez utiliser pour résoudre les problèmes détectés.

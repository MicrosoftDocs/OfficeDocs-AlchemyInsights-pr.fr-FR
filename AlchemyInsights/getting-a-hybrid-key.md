# <a name="getting-an-exchange-hybrid-key"></a>Obtention d’une clé hybride Exchange

Pour connecter votre organisation locale à Exchange Online dans Microsoft 365, vous devez configurer au moins un serveur Exchange dans votre organisation locale. Ce serveur est utilisé pour créer un déploiement hybride qui connecte votre organisation Exchange locale à Exchange Online. Pour obtenir une clé hybride pour ce serveur, suivez les étapes de l' [Assistant distribution de clé de produit hybride Exchange](https://aka.ms/hybridkey).

# <a name="getting-an-exchange-hybrid-key"></a>Obtention d’une clé d’échange hybride

Pour vous connecter à votre organisation locale vers Exchange Online dans Office 365, vous devez configurer au moins un serveur Exchange dans votre organisation locale. Ce serveur est utilisé pour créer un déploiement hybride qui connecte votre organisation d’Exchange local vers Exchange Online. Pour obtenir une clé hybride pour ce serveur, suivez les étapes de [l’Assistant de distribution de clés du produit Exchange hybride](http://aka.ms/hybridkey).

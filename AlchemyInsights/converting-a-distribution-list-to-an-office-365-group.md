# <a name="convert-a-distribution-group-to-a-microsoft-365-group"></a>Convertir un groupe de distribution en groupe Microsoft 365

Vous pouvez convertir des groupes de distribution en Groupes Microsoft 365 s'ils répondent à certains critères. Pour convertir un groupe ou résoudre les problèmes de conversion, voir Mettre à niveau les listes de distribution vers [Groupes Microsoft 365 dans Outlook.](https://docs.microsoft.com/microsoft-365/admin/manage/upgrade-distribution-lists)

W alsoe have a script you can run to help you with [Validating Distribution Group eligibility for upgrade to Microsoft 365 Group](https://aka.ms/DLToM365Group).

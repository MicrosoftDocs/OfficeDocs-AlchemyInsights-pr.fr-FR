# <a name="convert-a-distribution-group-to-a-microsoft-365-group"></a>Convertir un groupe de distribution en groupe Microsoft 365

Vous pouvez convertir les groupes de distribution en groupes Microsoft 365 s’ils répondent à certains critères. Pour convertir un groupe ou résoudre les problèmes de conversion, voir [mettre à niveau les listes de distribution vers des groupes Microsoft 365 dans Outlook](https://docs.microsoft.com/microsoft-365/admin/manage/upgrade-distribution-lists).

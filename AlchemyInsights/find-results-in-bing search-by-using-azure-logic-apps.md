# <a name="find-results-in-bing-search-by-using-azure-logic-apps"></a>Rechercher des résultats dans Bing Search à l’aide d’Azure Logic Apps

En utilisant Bing Search à partir d’une application logique, vous pouvez automatiser le traitement des résultats de recherche et les rendre disponibles pour d’autres actions. Par exemple, vous pouvez trouver des éléments d’actualités, puis vous pouvez les publier sur Twitter en tant que tweets dans votre flux.

Pour en savoir plus, voir Résultats de la recherche [Bing à l’aide d’Azure Logic Apps.](https://go.microsoft.com/fwlink/?linkid=2151928)

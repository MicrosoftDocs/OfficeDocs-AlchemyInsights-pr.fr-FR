# <a name="remove-a-hub-site"></a>Supprimer un site hub

Annuler l’inscription d’un site hub dans le nouveau centre d’administration de SharePoint
  
1. Dans le nouveau centre d’administration de SharePoint, sous **Sites**, cliquez sur **sites actifs**. 
    
2. Sélectionnez le site, cliquez sur **un site Hub**, puis cliquez sur **Annuler l’inscription en tant que site hub**. 
    
3. Cliquez sur **OK**. 
    
[Pour plus d’informations sur les sites hub](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f?ui=en-US&amp;rs=en-US&amp;ad=US)

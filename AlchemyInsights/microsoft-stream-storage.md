# <a name="microsoft-stream-storage"></a>Stockage Microsoft Stream

Les clients Microsoft Stream bénéficient d’une allocation fixe de 500 Go de stockage et d’un espace de stockage supplémentaire de 0,5 Go par utilisateur sous licence.
Pour plus d’informations sur le stockage, voir :

- [Vue d’ensemble du stockage de flux Microsoft](https://docs.microsoft.com/stream/license-overview#storage)
- [Ajouter du stockage à Microsoft Stream](https://docs.microsoft.com/stream/storage-add-on)
- [Quotas et limitations dans Microsoft Stream](https://docs.microsoft.com/stream/quotas-and-limitations)

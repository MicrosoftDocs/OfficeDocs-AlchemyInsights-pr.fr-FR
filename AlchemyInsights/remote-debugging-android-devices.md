# <a name="use-microsoft-edge-to-remotely-debug-an-android-device-from-a-development-computer"></a>Utiliser Microsoft Edge pour déboguer un appareil Android à distance à partir d’un ordinateur de développement

Pour déboguer à distance le contenu actif sur un appareil Android à partir de votre ordinateur :

1. Configurez l’appareil Android pour le débogage distant et découvrez-le à partir de l’ordinateur de développement.
2. À partir de l’ordinateur de développement, inspectez et déboguez le contenu réel sur l’appareil Android.
3. Capture de contenu en direct de l’appareil Android sur une instance DevTools sur l’ordinateur de développement.

# <a name="prepaid-renewal"></a>Renouvellement prépayé

1. Dans le Centre d'administration, accédez à la page [Abonnements](https://go.microsoft.com/fwlink/p/?linkid=842054), ou choisissez **Facturation** \> **Abonnements**.
    
2. Choisissez l’abonnement prépayé que vous souhaitez étendre, puis cliquez sur **étendre**.
    
3. Choisissez l’option de **clé de produit**. (S’il s’agissait d’une clé de produit achetée à un emplacement de vente au détail, peut avoir l’option d’extension avec d’autres méthodes de paiement.)
    
4. Entrez la clé de produit.
    
5. Assurez-vous que les informations sont correctes, puis cliquez sur **suivant**.
    
6. Choisissez le bouton radio pour étendre la date de fin des licences existants, puis cliquez sur **suivant**.

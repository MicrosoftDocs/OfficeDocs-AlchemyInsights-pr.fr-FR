# <a name="limit-sharing-in-microsoft-365"></a>Limiter le partage dans Microsoft 365

Même si vous ne pouvez pas désactiver le partage interne ou si vous supprimez le bouton partager de sites, vous pouvez limiter le partage dans Microsoft 365 pour répondre aux besoins de votre organisation. Pour plus d’informations, consultez la rubrique [limiter le partage dans Microsoft 365](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing).

# <a name="set-up-a-conferencing-bridge-for-teams"></a>Configurer un pont de conférence pour Teams

Il vous suffit de configurer la fonctionnalité de audioconférence pour les utilisateurs qui comptent planifier ou organiser des réunions. Nous avons mis en place un processus étape par étape pour effectuer la configuration du système téléphonique. Veuillez consulter [La configuration système téléphonique de votre organisation](https://docs.microsoft.com/MicrosoftTeams/phone-number-calling-plans/port-order-overview).

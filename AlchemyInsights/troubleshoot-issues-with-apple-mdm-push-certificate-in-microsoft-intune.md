# <a name="troubleshoot-issues-with-apple-mdm-push-certificate-in-microsoft-intune"></a>Résoudre les problèmes de certificat de Push Apple Mobile Device Manager dans Microsoft Intune

Passez en revue les ressources répertoriées ci-dessous pour résoudre le problème maintenant. 
  
1. Passez en revue [ce document](https://docs.microsoft.com/intune/apple-mdm-push-certificate-get) pour configurer ou renouveler le certificat d’émetteur Mobile Device Manager Apple. 
    
2. Utiliser le [Portail de résolution des problèmes de Intune](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) pour diagnostiquer et résoudre les échecs d’inscription courantes. Passez en revue [ce document](https://docs.microsoft.com/intune/help-desk-operators) pour plus d’informations. 
    
3. Passez en revue ces documents pour la liste des erreurs qui empêchent l’inscription et les résolutions à chacun : [guide de résolution des problèmes](https://support.microsoft.com/help/4039809/troubleshooting-ios-device-enrollment-in-intune) et [dépannage doc](https://docs.microsoft.com/intune-classic/troubleshoot/troubleshoot-device-enrollment-in-intune).

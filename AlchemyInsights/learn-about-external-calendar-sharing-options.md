# <a name="learn-about-external-calendar-sharing-options"></a>En savoir plus sur les options de partage de calendrier externe

Si vous devez planifier des réunions avec des personnes extérieures à votre organisation, Microsoft 365 vous permet de mettre des calendriers à disposition des « utilisateurs externes », qui doivent voir les informations de disponibilité mais pas de comptes d’utilisateur pour votre environnement Microsoft 365. Pour plus d’informations, consultez la rubrique [partager des calendriers avec des utilisateurs externes](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users).

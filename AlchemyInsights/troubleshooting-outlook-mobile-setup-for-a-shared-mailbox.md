# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Résolution des problèmes d’installation d’Outlook Mobile pour une boîte aux lettres partagée

Vous ne pouvez pas accéder à la boîte aux lettres partagée à partir de l'application Outlook installée sur l'appareil mobile. Voulez-vous utiliser cette fonctionnalité ? Informer l’équipe d’ingénierie Outlook. [Votez ici](https://go.microsoft.com/fwlink/?linked=862116)!
  
Vous pouvez uniquement utiliser une boîte aux lettres partagée sur un appareil mobile en ouvrant votre navigateur, en vous connectant, puis en accédant à Outlook sur le Web. Pour obtenir des instructions, consultez [la rubrique ajouter une boîte aux lettres partagée dans Outlook sur le Web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). À partir d’Outlook sur le web, vous pouvez accéder à la boîte aux lettres partagée.
  
En savoir plus : [obtenir de l’aide dans l’application pour Outlook pour iOS et Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)

# <a name="move-email-to-microsoft-365"></a>Déplacer le courrier électronique vers Microsoft 365

Vous disposez de plusieurs options pour migrer le courrier électronique vers Microsoft 365. Le [conseiller de migration de courrier 365 de Microsoft](https://aka.ms/alchemyinsight-mailmigrationadvisor) vous aide à choisir le meilleur pour votre environnement et vous guide tout au long de la migration du courrier.
  
Pour en savoir plus sur vos options sans démarrer la migration, consultez la rubrique [décider d’un chemin de migration](https://docs.microsoft.com/Exchange/mailbox-migration/decide-on-a-migration-path).

Consultez également la page [adopter Microsoft 365 avec FastTrack](https://www.microsoft.com/fasttrack/microsoft-365/office-365) pour obtenir de l’aide sur la migration, si éligible.

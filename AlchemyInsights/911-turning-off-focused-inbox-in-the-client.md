# <a name="turn-off-focused-inbox-in-email-clients"></a>Désactiver la boîte de réception destinées dans les clients de messagerie

Pour désactiver la fonction axés sur la boîte de réception dans Outlook 2016, Outlook sur le web, Outlook.com/Hotmail.com, 2016 Outlook pour Mac ou de messagerie pour Windows 10, voir [désactiver axés sur la boîte de réception](https://support.office.com/article/f714d94d-9e63-4217-9ccb-6cb2986aa1b2.aspx).
  
Pour désactiver la boîte de réception destinées pour tout le monde dans l’organisation, voir [Configurer axés sur la boîte de réception tout le monde dans votre organisation](https://support.office.com/article/613a845c-4b71-41de-b331-acdcf5b6625d.aspx).

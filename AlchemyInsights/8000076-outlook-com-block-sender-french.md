# <a name="bloquer-ou-dbloquer-expditeurs-dans-outlookcom"></a>Bloquer ou débloquer expéditeurs dans Outlook.com

Si le recevez du courrier indésirable, vous faire pouvez bloquer les adresses de messagerie et des domaines que le ne voulez pas recevoir des messages. Ajouter une adresse de messagerie ou un domaine à votre liste des expéditeurs bloqués pour déplacer les messages vers le dossier courrier indésirable.

1. **Options**accédez à **paramètres** > .
2. Dans le volet, sélectionnez **courrier indésirable** > **expéditeurs bloqués**.
3. Entrez l'adresse de messagerie dans les **expéditeurs bloqués**, sélectionnez **Ajouter**.
4. **Enregistrement**sélectionnez.

Pour plus d'Lisez, [bloquer ou débloquer expéditeurs dans Outlook.com](https://support.office.com/fr-fr/article/bloquer-ou-débloquer-expéditeurs-dans-outlook-com-afba1c94-77bb-4f50-8b85-057cf52f4d5e?ui=fr-FR&rs=fr-FR&ad=FR).

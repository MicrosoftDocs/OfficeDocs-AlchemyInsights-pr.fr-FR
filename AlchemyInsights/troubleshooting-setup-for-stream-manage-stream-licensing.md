# <a name="managing-microsoft-stream-licenses"></a>Gestion des licences de flux Microsoft

**Important:** Vous devez être dans le rôle d’administrateur client pour effectuer les opérations décrites dans cette rubrique.

Vous devez être un administrateur général ou un administrateur de gestion des utilisateurs. Pour plus d’informations, consultez la rubrique [à propos des rôles d’administrateur Office 365](https://support.office.com/article/About-Office-365-admin-roles-da585eea-f576-4f55-a1e0-87090b6aaa9d).

Vous pouvez attribuer [des licences à des comptes d’utilisateur avec Office 365 PowerShell](https://go.microsoft.com/fwlink/p/?linkid=850410).

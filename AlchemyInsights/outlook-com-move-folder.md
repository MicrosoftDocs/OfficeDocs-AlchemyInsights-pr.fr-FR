# <a name="move-a-folder"></a>Déplacer un dossier

Vous pouvez faire glisser des dossiers sous **favoris** pour les réorganiser.
  
Vous ne pouvez pas réorganiser les dossiers sous **dossiers** ou déplacer un sous-dossier pour le transformer en dossier.

*Remarque:* Si vous déplacez un dossier pour qu’il devienne un sous-dossier, créez un nouveau dossier et copiez tous les messages électroniques du sous-dossier vers le nouveau dossier.
  
Pour en savoir plus, consultez la rubrique [utilisation des dossiers de messages dans Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a).

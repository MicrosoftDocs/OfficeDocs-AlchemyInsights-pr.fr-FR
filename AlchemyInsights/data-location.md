# <a name="data-location"></a>Emplacement des données

Vous pouvez afficher l’emplacement de votre client dans le centre d’administration en accédant à la [page **Profil** ](https://admin.microsoft.com/AdminPortal/Home#/Settings/OrganizationProfile) de l’organisation et en sélectionnant **emplacement des données**.

Pour les applications auxquelles vous n’êtes pas abonné, consultez [l’emplacement de stockage de vos données client Microsoft 365](https://docs.microsoft.com/office365/enterprise/o365-data-locations).

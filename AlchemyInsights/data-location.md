# <a name="data-location"></a>Emplacement des données

Nous maintenir plusieurs copies de vos données dans les centres de données pour la redondance. [Voir où se trouve vos données](https://office.com/datamaps). Si nous développer l’emplacement de vos données dans un nouveau pays ou la région, nous vous donnerons préavis d’un mois.
  
Pour plus d’informations sur la sécurité Office 365, de confidentialité et de conformité, voir le [Centre de gestion de la confidentialité Office 365](https://products.office.com/business/office-365-trust-center-welcome).

# <a name="data-location"></a>Emplacement des données

Nous assurons la redondance de plusieurs copies de vos données dans tous les centres de données. [Voir où se trouvent vos données](https://office.com/datamaps). Si nous développons l’emplacement de vos données dans un nouveau pays ou une nouvelle région, nous vous fournirons un préavis d’un mois.
  
Pour plus d’informations sur la sécurité, la confidentialité et la conformité d’Office 365, consultez le centre de gestion de la confidentialité d' [office 365](https://products.office.com/business/office-365-trust-center-welcome).

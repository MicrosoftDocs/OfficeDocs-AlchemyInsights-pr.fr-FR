# <a name="learn-how-to-configure-agenda-settings"></a>Découvrez comment configurer les paramètres de l’ordre du jour

Paramètres de l’ordre du jour sont situés dans les paramètres du **calendrier** dans Outlook sur le web. Pour plus d’informations, consultez [paramètres du calendrier dans Outlook sur le web](https://support.office.com/article/12cba5a4-4f95-4d00-bfc3-b694aa67ac8f).
  
Administrateurs peuvent utiliser l’applet de commande **Set-CalendarNotification** dans Exchange Online PowerShell pour configurer les paramètres de l’ordre du jour. Pour plus d’informations, voir [Set-CalendarNotification](https://technet.microsoft.com/library/dd351284).

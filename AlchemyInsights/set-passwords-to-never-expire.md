# <a name="set-passwords-to-never-expire"></a>Définir les mots de passe de sorte qu'ils n'expirent jamais

Définissez les mots de passe de sorte qu'ils n'expirent jamais à partir de la [ &amp; confidentialité](https://portal.office.com/adminportal/home#/settings/security). Cliquez sur «modifier» en regard de stratégie de mot de passe, puis faites basculer le curseur sur «activé».
  
Pour plus d'informations, consultez la rubrique relative à [la définition de la stratégie d'expiration de mot de passe pour votre organisation](https://support.office.com/article/set-the-password-expiration-policy-for-your-organization-0f54736f-eb22-414c-8273-498a0918678f).

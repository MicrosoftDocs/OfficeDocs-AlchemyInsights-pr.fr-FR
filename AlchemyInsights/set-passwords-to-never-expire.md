# <a name="set-passwords-to-never-expire"></a>Mots de passe n’expire jamais

Définir les mots de passe n’expire jamais à partir de [sécurité &amp; confidentialité](https://portal.office.com/adminportal/home#/settings/security). Cliquez sur « Modifier » en regard de la stratégie de mot de passe et activer le curseur sur « on ».
  
Pour plus d’informations, voir : [définir la stratégie d’expiration de mot de passe pour votre organisation](https://support.office.com/article/set-the-password-expiration-policy-for-your-organization-0f54736f-eb22-414c-8273-498a0918678f).

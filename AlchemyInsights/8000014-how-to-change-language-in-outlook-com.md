# <a name="change-your-language"></a>Changer de langue

1. Dans le coin supérieur droit, sélectionnez **paramètres**![paramètres](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png) \> **Options**.
    
2. Sous **Options**, sélectionnez **Général** \> **région et le fuseau horaire**.
    
3. Sous **paramètres de région et fuseaux horaires**, choisissez votre langue.
    
4. Sélectionnez **Enregistrer**.
    
Pour plus d’informations, voir [Modifier le fuseau horaire et les paramètres de langue dans Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873132).

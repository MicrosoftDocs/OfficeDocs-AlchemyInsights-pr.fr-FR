# <a name="change-organization-profile"></a>Modifier le profil d’organisation

Modifiez le nom de votre société, votre adresse, les coordonnées techniques, et plus encore dans le centre d’administration sur la page Paramètres de l'[organisation](https://go.microsoft.com/fwlink/p/?linkid=2067339) des **paramètres** > .

Veillez à renseigner tous les champs obligatoires marqués avec un astérisque (*) pour pouvoir enregistrer vos modifications.

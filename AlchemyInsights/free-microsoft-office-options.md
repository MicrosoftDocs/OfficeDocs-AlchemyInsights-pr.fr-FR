# <a name="free-microsoft-office-options"></a>Options Microsoft Office gratuites

Vous pouvez créer, partager et collaborer gratuitement avec les versions en ligne de Microsoft Word, PowerPoint, Excel et OneNote.

Vous pouvez également utiliser les options gratuites suivantes :

- [Applications Microsoft Office pour appareil mobile](https://products.office.com/mobile/office?wt.mc_id=Cons_Office_Chatbot). Utilisez les applications Office pour les appareils mobiles (iPhone, iPad, téléphones Android et tablettes Android) avec un écran dont la taille est inférieur à 10,1 Po.

- Les étudiants des établissements éligibles peuvent [se connecter à un compte Microsoft 365 Éducation gratuitement](https://www.microsoft.com/education/products/office?wt.mc_id=Cons_Office_Chatbot).

- Certains nouveaux ordinateurs incluent une année gratuite d’Office. Pour plus d’informations, consultez [Activer un achat ou une offre gratuite d’Office sur un nouveau PC](https://support.office.com/article/89881633-0b26-4ca8-816b-93f347bd92c0?wt.mc_id=Cons_Office_Chatbot).

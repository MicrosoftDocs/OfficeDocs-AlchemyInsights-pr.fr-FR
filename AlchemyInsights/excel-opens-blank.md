# <a name="excel-opens-blank"></a>Excel s’ouvre vide

Si Excel démarre mais affiche un écran vide à l’endroit où vous souhaitez que le fichier s’affiche, consultez la rubrique [résolution des problèmes d’ouverture d’Excel lorsque vous double-cliquez sur une icône de fichier ou un nom de fichier](https://docs.microsoft.com/office/troubleshoot/excel/excel-opens-blank).

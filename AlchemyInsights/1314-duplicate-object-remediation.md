# <a name="duplicate-object-synchronization-errors"></a>Erreurs de synchronisation de copie de l’objet

Si vous recevez l’erreur suivante lors d’une synchronisation :
  
- Un objet synchronisé avec la même adresse proxy existe déjà dans votre annuaire Microsoft Online Services.
    
- Un message similaire pour UserPrincipalName.
    
Utilisez l’intégrité AADConnect nouveau diagnostic pour résoudre les erreurs de copie de l’objet. Pour plus d’informations, voir [diagnostiquer et résoudre les erreurs de synchronisation attribut dupliqué](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).

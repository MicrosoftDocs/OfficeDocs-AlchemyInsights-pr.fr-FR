# <a name="how-to-sign-out-of-outlook-on-the-web"></a>Procédure de déconnexion d’Outlook sur le Web

1. Sélectionnez votre nom ou votre image dans le coin supérieur droit.
    
    Si vous ne voyez pas votre image en haut de l’écran, vérifiez si le blocage des publicités est activé. Parfois, cela empêche l’affichage de l’image.
    
2. Sélectionnez **déconnexion**. 
    
    Pour arrêter de se connecter automatiquement à votre compte professionnel ou scolaire, désactivez la case à cocher **maintenir la connexion** sur la page où vous entrez votre mot de passe. 
    
Pour en savoir plus, consultez [la rubrique Comment se connecter à Outlook sur le Web ou s’en](https://support.office.com/article/763fab4d-0138-4814-b450-37fc286bcb79)déconnecter.

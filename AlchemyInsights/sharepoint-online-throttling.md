# <a name="sharepoint-online-throttling"></a>SharePoint Online limitation

SharePoint Online utilise la limitation pour mettre à jour d’optimiser les performances et la fiabilité du service SharePoint Online. Le nombre d’actions de l’utilisateur ou simultanés limitations appelle (par script ou code) pour éviter la surutilisation des ressources. Si vous obtenir limitées, 99 % du temps, il s’agit en raison d’un code personnalisé.
  
Pour plus d’informations sur la limitation, voir [éviter l’obtention limitées ou bloqués dans SharePoint Online](https://go.microsoft.com/fwlink/?linkid=2022019)

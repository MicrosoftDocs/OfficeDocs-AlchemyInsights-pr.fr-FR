# <a name="replace-a-word-or-phrase"></a>Remplacer un mot ou une expression

Pour remplacer plusieurs instances d’un mot ou d’une expression dans un document Word :

1. Accédez à l' **Accueil**  >  **remplacer**ou appuyez sur Ctrl + H.
2. Dans la zone **Rechercher** , entrez le mot ou l’expression que vous souhaitez remplacer. 
3. Dans la zone **remplacer par** , entrez votre nouveau texte.
3. Sélectionnez **remplacer tout** pour modifier toutes les occurrences du mot ou de l’expression, ou cliquez sur **suivant** jusqu’à ce que vous trouviez le ou les un (s) que vous souhaitez mettre à jour, puis sélectionnez **remplacer**.

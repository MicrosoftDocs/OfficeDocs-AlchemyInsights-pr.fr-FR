# <a name="microsoft-stream-licensing-overview"></a>Vue d’ensemble des licences Microsoft Stream

Microsoft Stream est un service Cloud sous licence par abonnement utilisateur. Elle est disponible dans le cadre des abonnements Microsoft 365 admissibles. Consultez la rubrique tarifs Microsoft 365 pour obtenir des informations sur chaque type d’abonnement.

- [Vue d’ensemble des licences Microsoft Stream](https://docs.microsoft.com/stream/license-overview)

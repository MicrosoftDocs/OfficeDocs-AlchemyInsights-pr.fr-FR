# <a name="change-your-password"></a>Modifier votre mot de passe

Si vous avez oublié votre mot de passe Outlook.com, accédez à la [page de réinitialisation du mot de passe de compte Microsoft](https://go.microsoft.com/fwlink/p/?linkid=841909). Dans le cas contraire, suivez les instructions ci-dessous.
  
1. Sélectionnez votre nom ou votre image de profil dans le coin supérieur droit et sélectionnez **mon compte**. 
    
2. Sous votre nom, sélectionnez **actions supplémentaires**  >  **modifier le mot de passe**. 
    
3. Suivez les étapes pour signer et choisir un nouveau mot de passe. 
    
Pour en savoir plus, consultez la rubrique [modifier mon mot de passe dans Outlook.com](https://support.office.com/article/2138d690-811c-4545-b2f3-e4dbe80c9735.aspx).

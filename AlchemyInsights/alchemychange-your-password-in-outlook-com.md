# <a name="change-your-password"></a>Modifier votre mot de passe

Si vous avez oublié votre mot de passe, accédez au [compte Microsoft rétablir la page mot de passe](https://go.microsoft.com/fwlink/p/?linkid=841909). Dans le cas contraire, suivez les instructions ci-dessous.
  
1. Sélectionnez votre image de profil ou le nom dans le coin supérieur droit et sélectionnez **Afficher le compte**. 
    
2. Sous le Bonjour ! bannière, sélectionnez **Modifier mot de passe**. 
    
3. Choisissez un nouveau mot de passe et cliquez sur **Enregistrer**. 
    
Pour plus d’informations, consultez la rubrique [Changer mon mot de passe dans Outlook.com](https://support.office.com/article/2138d690-811c-4545-b2f3-e4dbe80c9735.aspx).

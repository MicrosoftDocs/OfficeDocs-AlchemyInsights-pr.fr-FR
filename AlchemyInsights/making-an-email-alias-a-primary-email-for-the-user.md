# <a name="make-an-email-alias-the-primary-address-for-a-user"></a>Transformer un alias de messagerie en adresse principale pour un utilisateur

Pour obtenir des instructions, consultez [la rubrique ajouter un autre alias de messagerie pour un utilisateur](https://docs.microsoft.com/office365/admin/email/add-another-email-alias-for-a-user).

**Remarque**: Si votre organisation est en mode hybride, vous devez modifier l’adresse de messagerie principale dans votre organisation Exchange locale. Pour plus d’informations, consultez [la rubrique ajouter ou supprimer des adresses de messagerie pour une boîte aux lettres](https://technet.microsoft.com/library/bb123794.aspx).

# <a name="change-your-password"></a>Modifier votre mot de passe

Votre mot de passe Outlook.com est identique à votre mot de passe du compte Microsoft.
  
Pour modifier votre mot de passe, accédez à [https://account.microsoft.com/security](https://go.microsoft.com/fwlink/p/?linkid=842325&amp;clcid=0x409) et sélectionnez **Modifier le mot de passe**. 
  
Si vous avez oublié votre mot de passe, accédez au [compte Microsoft rétablir la page mot de passe](https://go.microsoft.com/fwlink/p/?linkid=841909).
  
Pour plus d’informations, voir [modifier votre mot de passe dans Outlook.com](https://go.microsoft.com/fwlink/?linkid=873109).

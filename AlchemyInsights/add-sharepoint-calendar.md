# <a name="add-a-sharepoint-calendar"></a>Ajouter un calendrier SharePoint

Les calendriers SharePoint sont des types de liste classiques qui sont parfois confondus avec les calendriers de groupe Microsoft 365.
 
Pour créer un calendrier SharePoint, suivez les étapes suivantes :
 
1.  Sélectionnez **l'icône d'engrenage Paramètres,** puis **sélectionnez Ajouter une application.**
2.  Dans la **zone de recherche Rechercher une application,** tapez **Calendrier,** puis appuyez sur Entrée.
3.  Sélectionnez l'application **intitulée Calendrier,** puis entrez un nom pour votre calendrier.

Pour plus d'informations sur les groupes Microsoft 365, voir Groupes [Microsoft 365.](https://support.office.com/article/Learn-about-Office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2)

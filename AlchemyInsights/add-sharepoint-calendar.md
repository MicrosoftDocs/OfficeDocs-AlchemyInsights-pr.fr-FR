# <a name="add-a-sharepoint-calendar"></a>Ajouter un calendrier SharePoint

Les calendriers SharePoint sont des types de liste classiques qui sont parfois confondus avec les calendriers de groupe Microsoft 365.
 
Pour créer un calendrier SharePoint, procédez comme suit :
 
1.  Sélectionnez l’icône d’engrenage **paramètres** , puis sélectionnez **Ajouter une application**.
2.  Dans la zone **Rechercher une application** de recherche, tapez **calendrier** , puis appuyez sur entrée.
3.  Sélectionnez l’application nommée **calendrier**, puis entrez un nom pour votre calendrier.

Pour plus d’informations sur les groupes Microsoft 365, consultez la rubrique [en savoir plus sur les groupes microsoft 365](https://support.office.com/article/Learn-about-Office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2).

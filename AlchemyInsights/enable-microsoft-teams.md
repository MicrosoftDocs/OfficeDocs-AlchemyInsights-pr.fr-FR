# <a name="enable-and-use-microsoft-teams"></a>Activer et utiliser Microsoft teams

Si vous n’avez pas d’accès à Microsoft Teams, consultez [la rubrique Comment obtenir l’accès à Microsoft teams](https://support.office.com/article/How-do-I-get-access-to-Microsoft-Teams-fc7f1634-abd3-4f26-a597-9df16e4ca65b.aspx).

Si vous avez déjà Microsoft teams et que vous avez besoin de savoir comment le configurer et le gérer, consultez [la rubrique gérer les paramètres de Microsoft teams pour votre organisation](https://docs.microsoft.com/MicrosoftTeams/enable-features-office-365).

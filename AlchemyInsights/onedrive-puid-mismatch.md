# <a name="onedrive-puid-mismatch"></a>Non-concordance des PUID OneDrive
Il s’avère que vous rencontrez une [incompatibilité PUID](https://docs.microsoft.com/sharepoint/support/administration/access-denied-or-need-permission-error-sharepoint-online-or-onedrive-for-business#when-accessing-a-onedrive-site) dans OneDrive entreprise. Suivez les étapes ci-dessous pour que nous puissions vous aider dès que possible :

1. Cliquez sur **contacter le support**, puis sur **nouvelle demande de service**.
2. Laissez le titre et la description comme non- **concordance PUID sur le site OneDrive**.
3. Une fois le ticket soumis, veuillez le mettre à jour avec les informations suivantes :
    - URL du site OneDrive concernée.
    - Nom d’utilisateur principal (UPN) de l’utilisateur concerné.

# <a name="enable-versioning-for-a-sharepoint-list-or-library"></a>Activer le contrôle de version pour une liste ou une bibliothèque SharePoint

Le contrôle de version, combiné à d’autres paramètres, tels que le retrait, vous permet de contrôler de manière importante le contenu publié sur votre site et peut fournir une valeur réelle si vous avez besoin de consulter ou de restaurer une ancienne version d’un élément ou d’un fichier.</span></p> <p><span style="mso-bidi-font-family: Calibri;

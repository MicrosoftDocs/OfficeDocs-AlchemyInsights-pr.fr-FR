# <a name="a-user-has-two-mailboxes"></a>Un utilisateur dispose de deux boîtes aux lettres

Les environnements hybrides qui utilisent Azure Active Directory Connect (AAD Connect) ou DirSync peuvent accidentellement faire en sorte qu’un utilisateur dispose de deux boîtes aux lettres: une dans l’organisation locale et une autre dans le Cloud.

Pour résoudre ce problème, reportez-vous à la rubrique [Effacer définitivement les informations de boîte aux lettres précédentes](https://blogs.technet.microsoft.com/exchange/2018/01/17/permanently-clear-previous-mailbox-info/).

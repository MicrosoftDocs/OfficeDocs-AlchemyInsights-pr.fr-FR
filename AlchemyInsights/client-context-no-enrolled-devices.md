# <a name="client-context---no-enrolled-devices"></a>Contexte client-aucun appareil d’enregistrement

Votre client ne dispose d’aucun appareil de gestion des appareils mobiles avec Intune. Si vous souhaitez inscrire et gérer vos appareils via Intune MDM, consultez la documentation [ici](https://docs.microsoft.com/intune/device-enrollment) pour obtenir la plateforme souhaitée. Si vous utilisez une seule fonctionnalité de protection de l’application ou d’une tierce partie, aucune action n’est nécessaire.

# <a name="problems-setting-up-an-exchange-email-account-in-outlook"></a>Problèmes de configuration d’un compte de messagerie Exchange dans Outlook

Si vous rencontrez des problèmes pour configurer un compte Exchange dans Outlook, vous pouvez exécuter l' [outil de configuration du compte Outlook](https://aka.ms/SaRA-OutlookSetupProfile) sur l’ordinateur concerné. Il exécute plusieurs vérifications et fournissent des solutions possibles à votre problème de configuration de compte. 
  
Si vous rencontrez encore des problèmes, consultez [Qu'outlook ne peut pas définir un nouveau profil à l’aide de découverte automatique Exchange pour une boîte aux lettres Exchange Online dans Office 365](https://support.microsoft.com/help/2404385/outlook-can-t-set-up-a-new-profile-by-using-exchange-autodiscover-for).

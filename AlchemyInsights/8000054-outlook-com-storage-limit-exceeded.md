# <a name="storage-limits-in-outlookcom"></a>Limites de stockage dans Outlook.com

Microsoft propose 15 Go d’espace de stockage de courrier électronique par compte Outlook.com gratuit. Accueil Office 365 et Office 365 personnelles abonnés obtenir 50 Go d’espace.
  
Si votre boîte de réception est complète, vous ne pourrez pas envoyer ou recevoir des messages. En outre, les personnes qui vous envoyer reçoit un message d’erreur que votre boîte de réception est plein. Pour libérer de l’espace de stockage dans votre boîte de réception, voir [limites de stockage dans Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=2001900&amp;clcid=0x409).

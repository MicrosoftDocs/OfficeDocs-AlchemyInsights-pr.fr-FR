# <a name="unable-to-delete-photos-from-files"></a>Impossible de supprimer les photos à partir de fichiers

Pour supprimer les fichiers ou les photos :
  
1. Accédez à votre dossier **boîte de réception** ou **Éléments envoyés** et supprimer le message électronique auxquels ils sont attachés à. 
    
2. Supprimer les éléments de vos **éléments supprimés**. 
    
Pour plus d’informations, consultez [la suppression des fichiers ou des photos dans Outlook.com](https://support.office.com/article/bae0531f-040f-4c42-90b9-786ca718c16d.aspx).

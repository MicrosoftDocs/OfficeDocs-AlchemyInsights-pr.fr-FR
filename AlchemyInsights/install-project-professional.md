# <a name="install-project-professional"></a>Installer Project Professionnel

Pour plus d’informations sur l’installation de la version d’abonnement de Project Online et de la version sans abonnement de Project 2019, 2016 ou 2013, voir [Installer Project](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281?ui=en-US&rs=en-US&ad=US) . 

Pour le projet 2010 ou 2007, suivez les étapes décrites dans [Office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) ou [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657). 

Le projet est vendu en tant qu’application autonome, donc si vous ne l’avez pas encore fait, [achetez ou essayez Project](https://products.office.com/project).

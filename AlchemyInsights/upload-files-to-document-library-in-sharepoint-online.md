# <a name="upload-files-to-onedrive-or-sharepoint"></a>Télécharger des fichiers vers OneDrive ou SharePoint

Vous pouvez télécharger des fichiers dans une bibliothèque de documents dans SharePoint Online en les faisant simplement glisser de votre ordinateur et en les déplaçant dans la bibliothèque de documents. Avec OneDrive, vous pouvez synchroniser des fichiers entre votre ordinateur et le Cloud, de sorte que vous puissiez accéder à vos fichiers depuis n’importe quel emplacement. Vous pouvez également utiliser vos fichiers synchronisés directement dans l’Explorateur de fichiers et accéder à vos fichiers même lorsque vous êtes en mode hors connexion. Pour plus d’informations sur le chargement et la synchronisation, consultez les liens ci-dessous.

# <a name="manage-a-group-without-an-exchange-license"></a>Gérer un groupe sans licence Exchange

Si votre compte d’administrateur n’a pas de boîte aux lettres, vous pouvez obtenir une erreur concernant une licence de produit manquante lorsque vous essayez de modifier un groupe Microsoft 365.
  
Étant donné que vous êtes l’administrateur client, vous pouvez modifier un groupe même si vous n’avez pas de licence Exchange. Accédez simplement aux [Exchange Admin Center](https://outlook.office365.com/ecp.aspx) \> **groupes** du centre d’administration Exchange pour gérer le groupe Microsoft 365.
  
Si vous souhaitez modifier des groupes dans le centre d’administration, ajoutez une licence Exchange à votre compte d’administrateur. Vous devez patienter jusqu’à ce que la boîte aux lettres soit entièrement créée avant de pouvoir modifier un groupe.

# <a name="print-results-in-a-summary-of-responses"></a>Imprimer les résultats dans un résumé des réponses

Vous pouvez [imprimer les résultats de Microsoft Forms](https://support.office.com/article/print-a-form-22100b98-ba3c-41c1-9513-f76caca664fc) en tant que Résumé des réponses ou pour un individu. 

Vous ne pouvez pas imprimer de formulaires sans réponse, mais vous pouvez demander ou voter sur cette fonctionnalité en visitant le [site Microsoft Forms UserVoice](https://microsoftforms.uservoice.com/forums/386451-welcome-to-microsoft-forms-suggestion-box).

# <a name="change-payment-method"></a>Changer votre mode de paiement

Vous pouvez vérifier le mode de paiement qui est configuré en accédant à la **facturation** \> **abonnements**. Choisissez l’abonnement que vous souhaitez vérifier. La méthode de paiement est configurée pour l’abonnement apparaît sous **Payment method**. Pour mettre à jour les détails de la méthode de paiement existant, cliquez sur **mettre à jour les informations de paiement**.
  
1. Dans le Centre d'administration, accédez à la page [Abonnements](https://go.microsoft.com/fwlink/p/?linkid=842054), ou choisissez **Facturation** \> **Abonnements**.
    
2. Choisissez l’abonnement que vous souhaitez modifier le mode de paiement pour, puis cliquez sur **Modifier le paiement**...
    
3. Suivez les étapes dans l’écran pour terminer la modification.
    
Pour obtenir des instructions étape par étape, voir : [changer votre mode de paiement pour Office 365 pour entreprises](https://support.office.com/article/8652f539-3123-4a8f-b9bd-6aa2f0e0372d)

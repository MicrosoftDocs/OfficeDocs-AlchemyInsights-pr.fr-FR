# <a name="get-started-with-the-developer-tools-in-microsoft-edge-chromium"></a>Prise en main des outils de développement dans Microsoft Edge (chrome)

Microsoft Edge a adopté le projet open-source de chrome, qui vise à faciliter la compatibilité du Web avec des plates-formes moins fragmentées et Web. Cette modification vous permettra d’utiliser Microsoft Edge pour créer et tester vos sites Web qui fonctionnent sur d’autres navigateurs en chrome.

Microsoft Edge (chrome) fournit également des [outils de développement de qualité](https://go.microsoft.com/fwlink/?linkid=2134941) pour vous être familier et compatible avec les outils que vous utilisez déjà.

Pour plus d’informations, consultez [la rubrique what’s New in the Microsoft Edge (chrome) devtools](https://go.microsoft.com/fwlink/?linkid=2135020).

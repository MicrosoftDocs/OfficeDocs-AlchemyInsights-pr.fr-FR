# <a name="address-book-in-outlookcom"></a>Carnet d'adresses dans Outlook.com

Votre carnet d'adresses Outlook.com est appelé personnes.

Pour accéder à votre page personnes, sélectionnez **personnes** dans le coin inférieur de la page, ou dans le sélecteur d'application dans le coin supérieur de la page.

Découvrez comment [créer, afficher et modifier des contacts dans Outlook.com](https://support.office.com/article/5b909158-036e-4820-92f7-2a27f57b9f01).

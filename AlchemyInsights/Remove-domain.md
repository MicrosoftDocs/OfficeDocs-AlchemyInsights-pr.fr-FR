# <a name="trying-to-remove-your-domain"></a>Vous essayez de supprimer votre domaine ?

Suivez les étapes décrites dans [Supprimer un domaine d’Office 365](https://docs.microsoft.com/office365/admin/get-help-with-domains/remove-a-domain).
  
Vous devrez mettre à jour les utilisateurs ou groupes qui sont affectés à ce nom de domaine avant de supprimer le domaine. Consultez le [Forum aux questions des domaines](https://docs.microsoft.com/office365/admin/setup/domains-faq) pour d’autres solutions courantes.

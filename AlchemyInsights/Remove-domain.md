# <a name="trying-to-remove-your-domain"></a>Vous essayez de supprimer votre domaine ?

Suivez les étapes décrites dans [Supprimer un domaine d’Office 365](https://support.office.com/article/Remove-a-domain-from-Office-365-f09696b2-8c29-4588-a08b-b333da19810c.aspx).
  
Vous devrez mettre à jour les utilisateurs ou groupes qui sont affectés à ce nom de domaine avant de supprimer le domaine. Consultez le [Forum aux questions des domaines](https://support.office.com/article/Domains-FAQ-1272bad0-4bd4-4796-8005-67d6fb3afc5a.aspx) pour d’autres solutions courantes.

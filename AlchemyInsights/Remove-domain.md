# <a name="trying-to-remove-your-domain"></a>Essayez de supprimer votre domaine ?

Pour supprimer votre domaine Office 365, suivez les étapes de [Supprimer un domaine à partir d’Office 365](https://support.office.com/article/Remove-a-domain-from-Office-365-f09696b2-8c29-4588-a08b-b333da19810c.aspx).
  
Vous devrez mettre à jour des utilisateurs ou des groupes qui sont affectés à ce nom de domaine avant de supprimer le domaine. Consultez le [Forum aux questions sur les domaines](https://support.office.com/article/Domains-FAQ-1272bad0-4bd4-4796-8005-67d6fb3afc5a.aspx) d’autres solutions courantes.

# <a name="onedrive-recycle-bin"></a>Corbeille OneDrive

Lorsque vous supprimez un élément ou un site de OneDrive, celui-ci n’est pas supprimé immédiatement. Les éléments supprimés vont dans la corbeille pendant un certain temps. Pendant ce temps, vous pouvez restaurer les éléments que vous avez supprimés dans leur emplacement d’origine. [En savoir plus](https://support.office.com/article/restore-deleted-files-or-folders-in-onedrive-949ada80-0026-4db3-a953-c99083e6a84f?ui=en-US&rs=en-US&ad=US).

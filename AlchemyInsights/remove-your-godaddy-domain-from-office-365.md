# <a name="remove-your-godaddy-domain-from-microsoft-365"></a>Supprimer votre domaine GoDaddy de Microsoft 365

Nous ne pouvons pas supprimer votre domaine GoDaddy de Microsoft 365 pour vous. Appelez le [service client GoDaddy](https://aka.ms/contact-godaddy) pour obtenir de l’aide.
  
Si le nom de l’organisation ne contient pas « NETORG » ou n’a pas été acheté auprès de GoDaddy, supprimez votre domaine en suivant ces étapes pour [supprimer un domaine de Microsoft 365.](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/remove-a-domain)

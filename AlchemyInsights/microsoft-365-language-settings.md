- Pour les utilisateurs, voir Modifier votre langue d'affichage et votre [fuseau horaire dans Microsoft 365 Entreprise pour toutes les applications.](https://support.microsoft.com/office/6f238bff-5252-441e-b32b-655d5d85d15b)

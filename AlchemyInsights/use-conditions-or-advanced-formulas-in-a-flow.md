# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>Utilisez les conditions ou les formules avancées dans un flux

Vous pouvez personnaliser un flux de sorte qu’il effectue des tâches uniquement si une condition est true. Pour obtenir des informations, voir [Ajouter une condition à un flux](https://go.microsoft.com/fwlink/?linkid=872112).
  
Pour créer une formule avancée, voir [schéma langage de définition du flux de travail pour les applications Azure logique](https://aka.ms/logicexpressions).

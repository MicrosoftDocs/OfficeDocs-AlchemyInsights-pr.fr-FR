# <a name="change-a-domain-name-letter-case-or-spelling"></a><span data-ttu-id="7ceb2-102">Modifier la casse ou l’orthographe d’une lettre de nom de domaine</span><span class="sxs-lookup"><span data-stu-id="7ceb2-102">Change a domain name letter case or spelling</span></span>

<span data-ttu-id="7ceb2-103">Pour modifier l’orthographe d’un nom de domaine personnalisé (par exemple, toutes les lettres minuscules ou majuscules et minuscules), le domaine doit être supprimé, puis ré-ajouté avec la casse ou l’orthographe souhaitée.</span><span class="sxs-lookup"><span data-stu-id="7ceb2-103">To change the spelling of a custom domain name, for example to all lowercase letters or a mix of lower and upper case letters, the domain must be removed and then re-added with the desired letter case or spelling.</span></span>

1. <span data-ttu-id="7ceb2-104">Dans la [page Domains (domaines)](https://portal.office.com/adminportal/home#/Domains), sélectionnez le domaine, puis **Supprimer le domaine**.</span><span class="sxs-lookup"><span data-stu-id="7ceb2-104">On the [Domains page](https://portal.office.com/adminportal/home#/Domains), choose the domain, and then select  **Remove domain**.</span></span></br>

    <span data-ttu-id="7ceb2-105">Pour obtenir des instructions plus détaillées, consultez [Comment supprimer un domaine](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/remove-a-domain?view=o365-worldwide).</span><span class="sxs-lookup"><span data-stu-id="7ceb2-105">For for more detailed instructions, see [How to remove a domain](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/remove-a-domain?view=o365-worldwide).</span></span>

2. <span data-ttu-id="7ceb2-106">Ajoutez à nouveau le domaine avec l'orthographe et la casse correctes en utilisant l’[Assistant Ajouter un domaine](https://portal.office.com/adminportal/home#/Domains/Wizard).</span><span class="sxs-lookup"><span data-stu-id="7ceb2-106">Re-add the domain with the correct spelling and letter case by using the [Add Domain Wizard](https://portal.office.com/adminportal/home#/Domains/Wizard).</span></span>

<span data-ttu-id="7ceb2-107">Vous ne pouvez pas modifier la casse des lettres ou renommer le domaine onmicrosoft.com ou un domaine acheté via Microsoft après l’inscription.</span><span class="sxs-lookup"><span data-stu-id="7ceb2-107">You can't change letter case or rename the onmicrosoft.com domain, or a domain purchased through Microsoft, after sign-up.</span></span>

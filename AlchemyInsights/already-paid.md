# <a name="already-paid"></a>Déjà payé ?

Si vous avez déjà envoyé le paiement, vous pouvez nous le faire comprendre :
  
1. Dans le Centre d’administration, accédez à la page **Facturation** \> [Abonnements](https://go.microsoft.com/fwlink/p/?linkid=842054).

2. Sélectionnez un abonnement, puis, dans la section **mon facture** , sélectionnez **Afficher Bill**.

3. Sur la page Détails de la **facture** , sélectionnez **déjà payé ?** Vous serez invité à entrer des informations relatives au paiement envoyé.

4. Une fois toutes les informations requises entrées, sélectionnez **Envoyer**.

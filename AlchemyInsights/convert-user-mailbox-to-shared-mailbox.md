# <a name="convert-a-user-mailbox-to-a-room-or-equipment-mailbox"></a>Convertir une boîte aux lettres de l’utilisateur à une boîte aux lettres de salle ou d’équipement

Vous pouvez également convertir une boîte aux lettres à partir d’une boîte aux lettres type à l’autre, mais vous devrez effectuer ces tâches à l’aide d’Exchange Online PowerShell. Consultez la rubrique [convertir une boîte aux lettres](https://go.microsoft.com/fwlink/p/?LinkId=832875) pour plus d’informations.

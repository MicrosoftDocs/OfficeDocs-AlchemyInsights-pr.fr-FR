# <a name="delete-a-communication-site"></a>Supprimer un site de communication

1. Accédez au site.
    
2. Cliquez sur l’icône Paramètres dans le coin supérieur droit.
    
3. Cliquez sur **paramètres du Site**. Sous **Informations de Site**, cliquez sur **Supprimer ce site**, puis cliquez sur **Supprimer**.

# <a name="delete-a-communication-site"></a>Supprimer un site de communication

1. Accédez au site.
    
2. Cliquez sur l’icône des paramètres dans le coin supérieur droit.
    
3. Cliquez sur **paramètres du site**. Sous **informations**sur le site, cliquez sur **supprimer ce site**, puis cliquez sur **supprimer**.

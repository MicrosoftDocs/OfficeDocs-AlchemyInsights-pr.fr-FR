# <a name="what-to-do-if-you-cant-create-a-team-in-microsoft-teams"></a>Que faire si vous ne pouvez pas créer une équipe dans Microsoft teams

Demandez à votre administrateur de teams d’effectuer les opérations suivantes :

1. Créez ou modifiez les paramètres du groupe de sécurité de l’organisation dans Microsoft 365.
2. Ajoutez-vous à ce groupe de sécurité.

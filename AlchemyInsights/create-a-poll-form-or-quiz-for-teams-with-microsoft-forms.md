# <a name="create-a-poll-form-or-quiz-for-teams-with-microsoft-forms"></a>Créer un sondage, un formulaire ou un questionnaire pour Teams avec Microsoft Forms

Vous voulez créer un formulaire, un sondage, une enquête ou un questionnaire dans Microsoft Teams ? Microsoft Forms peut vous aider !

 - [Activez ou désactivez Forms](https://support.office.com/article/turn-off-or-turn-on-microsoft-forms-8dcbf3ab-f2d6-459a-b8be-8d9892132a43) pour votre organisation ou activez-le uniquement pour des utilisateurs spécifiques.
 
 - [Créez, modifiez et collaborez](https://support.office.com/article/create-edit-and-collaborate-on-a-form-in-microsoft-teams-333b97a3-41d9-48bc-a1cb-84a96bd44e14) dans un formulaire dans Microsoft Teams.
 
 - [Forum aux questions](https://support.office.com/article/get-started-1dd58027-40dc-42d0-9ca4-80ddecc5c696) sur Microsoft Forms.

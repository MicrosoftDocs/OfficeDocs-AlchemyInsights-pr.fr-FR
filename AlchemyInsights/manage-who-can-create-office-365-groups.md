# <a name="manage-who-can-create-office-365-groups"></a>Gérer les personnes autorisées à créer des groupes Office 365

Par défaut, tout le monde dans votre organisation peut créer des groupes O365.
  
Si vous souhaitez modifier, vous pouvez créer un groupe de sécurité avec PowerShell qui limite la création de groupes à un sous-ensemble d’utilisateurs.
  
Suivez les instructions pas à pas : [gérer qui peut créer des groupes d’Office 365](https://support.office.com/article/4c46c8cb-17d0-44b5-9776-005fced8e618)

# <a name="manage-who-can-create-microsoft-365-groups"></a>Gérer les personnes autorisées à créer des groupes Microsoft 365

Par défaut, tous les utilisateurs de votre organisation peuvent créer des groupes O365.
  
Si vous souhaitez modifier cela, vous pouvez créer un groupe de sécurité avec PowerShell qui limite la création de groupe à un sous-ensemble d’utilisateurs.
  
Suivez les instructions pas à pas ici : gérer les [personnes autorisées à créer des groupes Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)

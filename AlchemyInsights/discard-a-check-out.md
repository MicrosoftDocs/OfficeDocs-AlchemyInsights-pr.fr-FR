# <a name="discard-a-check-out-from-a-document-library"></a>Annuler l’extraction d’une bibliothèque de documents

1. Cliquez sur le fichier que vous souhaitez annuler l’extraction.
    
2. Cliquez sur **plus**, puis cliquez sur **Annuler l’extraction**. 
    
3. Dans la boîte de dialogue, cliquez sur ** Discard Check Out ** pour effacer la case à cocher. 
    
Pour des questions supplémentaires sur l’archivage et extraction, voir les [questions fréquentes sur l’extraction, archivage et les versions](https://go.microsoft.com/fwlink/?linkid=2018786)

# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>Utilisation d’une boîte aux lettres partagée sur un appareil mobile

Pour pouvoir utiliser une boîte aux lettres partagée sur votre appareil mobile, vous devez [Télécharger Outlook Mobile](https://products.office.com/outlook-mobile-for-android-and-ios) , puis [Ajouter une boîte aux lettres partagée à Outlook Mobile](https://support.office.com/article/Add-a-shared-mailbox-to-Outlook-mobile-f866242c-81b2-472e-8776-6c49c5473c9f).
  
Pour obtenir de l’aide : obtenir de l' [aide dans l’application pour Outlook pour iOS et Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)

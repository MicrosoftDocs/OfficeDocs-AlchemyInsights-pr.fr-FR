# <a name="banned-sending-ip"></a>Adresse IP d’envoi interdit

Lors de l’envoi d’un message à Microsoft 365, un expéditeur peut être confronté à cette erreur. L’expéditeur doit visiter https://sender.office.com et suivre les instructions.  S’ils ne reçoivent pas de courrier électronique de vérification du portail, demandez-leur de vérifier le dossier de courrier indésirable et/ou d’essayer une autre adresse de messagerie.

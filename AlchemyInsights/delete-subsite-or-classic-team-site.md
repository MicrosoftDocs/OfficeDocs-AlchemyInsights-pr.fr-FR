# <a name="delete-a-classic-sharepoint-team-site-or-subsite"></a>Supprimer un site d’équipe SharePoint classique ou d’un sous-site

1. Accédez au site.
    
2. Cliquez sur l’icône **paramètres** dans le coin supérieur droit. 
    
3. Cliquez sur **paramètres du Site**. Sous **Actions du Site**, cliquez sur **Supprimer ce site**, puis cliquez sur **Supprimer**.

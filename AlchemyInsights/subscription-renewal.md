# <a name="subscription-recurring-billing"></a>Facturation périodique de l’abonnement

Si vous souhaitez connaître le moment où votre abonnement mensuel ou annuel sera facturé (si la **facturation périodique** est activée) ou expirer (si la **facturation périodique** est désactivée) :
  
1. Dans le centre d’administration, accédez à la page **facturation** \> **[de vos produits](https://go.microsoft.com/fwlink/p/?linkid=842054)** .

2. Pour connaître l’abonnement sur lequel vous souhaitez en savoir plus, reportez-vous à la date de **publication ou de** **fin** de la **facturation**.

4. Pour apporter une modification, sélectionnez **modifier** puis, dans le volet Paramètres de **facturation périodique** , sélectionnez **Désactiver** pour le désactiver. Si la facturation périodique est déjà désactivée, sélectionnez **Mensuelle** ou **Une fois** pour l’activer.

Pour en savoir plus, voir [Désactiver ou activer la facturation récurrente](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/renew-your-subscription).

# <a name="sharepoint-publishing-infrastructure"></a>Infrastructure de publication SharePoint


<p>SharePoint inclut un ensemble de fonctionnalités classiques qui vous permettent de créer et de publier des pages Web enrichies pour votre intranet. Activez l’infrastructure de publication SharePoint au niveau de la collection de sites pour les activer.</p>  <p>Pour plus d’informations, consultez les liens ci-dessous.</p>  <ul>  <li><a href="https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518">Activer les fonctionnalités de publication</a></li>  <li><a href="https://support.office.com/en-us/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553">Fonctionnalités de publication</a></li>  </ul>  <p>Remarques :</p>  <ul>  <li>Les modèles de site de publication classiques incluent les fonctionnalités de publication par défaut.</li>  <li>Les fonctionnalités de publication classiques ne sont pas disponibles sur les sites modernes. Les sites modernes fournissent une alternative plus récente par défaut.</li>  </ul>

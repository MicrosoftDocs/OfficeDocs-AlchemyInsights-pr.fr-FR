# <a name="refresh-using-web-connector-doesnt-work-properly"></a>L’actualisation à l’aide de Web Connector ne fonctionne pas correctement

Si vous avez un script de connecteur Web qui utilise la fonction [Web. page](https://msdn.microsoft.com/library/mt260924.aspx) et que vous avez mis à jour votre jeu de données ou votre rapport après le 18 novembre 2016, vous devez utiliser une passerelle pour que l’actualisation fonctionne correctement.

Pour plus d’informations, procédez comme suit :[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)

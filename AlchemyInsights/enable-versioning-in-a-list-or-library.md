# <a name="enable-versioning-for-a-list-or-library"></a>Activer le contrôle de version pour une liste ou une bibliothèque

1. Accédez à la liste ou bibliothèque.
    
2. Cliquez sur **paramètres**, puis cliquez sur **paramètres de la bibliothèque** ou **paramètres de la liste**.
    
3. Cliquez sur **paramètres de contrôle de version**.
    
4. Sélectionnez les paramètres souhaités, puis cliquez sur **OK**.

# <a name="site-is-not-available"></a>Le site n'est pas disponible

Si votre site SharePoint ou OneDrive n'est pas accessible à plusieurs utilisateurs, il peut s'agir d'un problème de service temporaire. Consultez le[tableau de bord d'État du service](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) pour voir si votre client est concerné.

# <a name="site-is-not-available"></a>Site n’est pas disponible

Si votre site SharePoint ou OneDrive n’est pas disponible à plusieurs utilisateurs, il peut être un problème temporaire de service. Vérifier[L’intégrité de Service du tableau de bord](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) pour voir si votre client est affecté.

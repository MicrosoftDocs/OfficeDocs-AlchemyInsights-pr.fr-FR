# <a name="im-not-seeing-the-calendars-i-expect-when-choosing-my-availability"></a>Je ne vois pas les calendriers attendus lors du choix de ma disponibilité

Si vous ne voyez pas les calendriers attendus, accédez à calendrier, appuyez sur le bouton de menu dans le coin supérieur gauche de l’écran et assurez-vous que les calendriers que vous souhaitez voir sont sélectionnés. Seuls les calendriers activés dans ce menu apparaissent lorsque vous choisissez des heures pour l’envoi de disponibilité.

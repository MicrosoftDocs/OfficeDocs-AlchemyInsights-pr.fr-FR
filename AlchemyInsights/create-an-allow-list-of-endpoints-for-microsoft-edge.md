# <a name="create-an-allow-list-of-endpoints-for-microsoft-edge"></a>Créer une liste verte de points de terminaison pour Microsoft Edge

Pour bénéficier de toutes les fonctionnalités de Microsoft Edge, vous devez être connecté à Internet. En outre, pour permettre au navigateur de communiquer via des pare-feu et d’autres mécanismes de sécurité sur Internet, vous devez ajouter un ensemble d’URL de domaine à sa [liste verte de points de terminaison](https://go.microsoft.com/fwlink/?linkid=2135054).

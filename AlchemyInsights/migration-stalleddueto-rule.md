# <a name="stalleddueto-errors-during-migration"></a>Erreurs StalledDueTo pendant la migration

Vous verrez peut-être plusieurs erreurs qui commencent par « StalledDueTo... » lors de la migration vers Exchange Online. Beaucoup de ces erreurs sont parfaitement normales. Pour plus d’informations, reportez-vous à la rubrique [limitation basée sur les ressources et définition des priorités dans les migrations Exchange Online](https://techcommunity.microsoft.com/t5/exchange-team-blog/resource-based-throttling-and-prioritization-in-exchange-online/ba-p/608020).

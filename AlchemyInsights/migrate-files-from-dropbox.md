# <a name="migrate-files-from-dropbox"></a>Migrer des fichiers à partir de Dropbox

Si vous migrez de Dropbox vers SharePoint Online, vous pouvez utiliser **Mover**.

Si vous souhaitez en savoir plus, veuillez consultez [Autorisation du connecteur Dropbox](https://docs.microsoft.com/sharepointmigration/mover-dropbox).

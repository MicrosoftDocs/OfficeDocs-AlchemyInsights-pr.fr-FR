# <a name="install-and-configure-onedrive"></a>Installer et configurer OneDrive

Le client de synchronisation OneDrive est préinstallé avec Windows 10. Pour installer OneDrive pour Windows 7 ou Mac, ou pour obtenir l’application mobile de votre téléphone ou votre tablette, accédez à [http://onedrive.com/download](http://onedrive.com/download).
  
Pour configurer le client de synchronisation OneDrive dans Windows, voir :
  
- [Configurer votre ordinateur pour synchroniser vos fichiers OneDrive Entreprise dans Office 365](https://go.microsoft.com/fwlink/?linkid=533375)
    
- [Synchroniser des fichiers SharePoint avec le nouveau client de synchronisation OneDrive](https://go.microsoft.com/fwlink/?linkid=871666)

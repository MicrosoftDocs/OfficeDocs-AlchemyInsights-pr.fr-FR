# <a name="edit-a-read-only-file"></a>Modifier un fichier en lecture seule

Dans les bibliothèques de documents, lorsque vous cliquez sur le nom d’un document, le programme Office peut brusquement ouvrez-le en tant qu’un fichier en lecture seule. Pour l’ouvrir pour modification, cliquez sur **Ouvrir** (trois points), puis cliquez sur **Modifier.**

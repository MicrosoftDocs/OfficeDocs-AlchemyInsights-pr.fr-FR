# <a name="edit-a-read-only-file"></a>Modifier un fichier en lecture seule

Dans les bibliothèques de documents, lorsque vous cliquez sur le nom d’un document, le programme Office peut l’ouvrir de manière inattendue en tant que fichier en lecture seule. Pour l’ouvrir en vue de le modifier, cliquez sur **ouvrir** (les trois points), puis cliquez sur **modifier.**

# <a name="cant-preview-all-of-an-attachment"></a>Impossible d’afficher l’aperçu de la pièce jointe

Nous avons pris conscience d’un problème : certains clients ne peuvent pas prévisualiser l’intégralité d’une pièce jointe. Pour résoudre ce problème, ouvrez la pièce jointe dans une nouvelle fenêtre.

# <a name="add-or-remove-an-email-address-for-a-user"></a>Ajouter ou supprimer une adresse de messagerie pour un utilisateur

[Pour commencer, cliquez ici](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions).

 **Remarque**: vous pouvez avoir plusieurs adresses de messagerie pour un utilisateur, mais des *alias de messagerie* supplémentaires ne peuvent être utilisés que pour la réception de messages électroniques. Si vous devez envoyer un message électronique avec un alias, créez une [boîte aux lettres partagée](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9). De cette façon, vous obtiendrez l’alias permettant d’envoyer et de recevoir des messages.

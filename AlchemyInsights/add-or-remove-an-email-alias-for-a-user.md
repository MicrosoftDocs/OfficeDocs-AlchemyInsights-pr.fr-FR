# <a name="add-or-remove-an-email-address-for-a-user"></a>Ajouter ou supprimer une adresse de messagerie pour un utilisateur

[Cliquez ici pour commencer](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions).
    
 **Remarque**: vous pouvez avoir plusieurs adresses de messagerie pour un utilisateur, mais supplémentaires *alias de messagerie* peuvent uniquement être utilisés pour la réception de courrier électronique. Si vous souhaitez envoyer un message électronique avec un alias, créer une [boîte aux lettres partagée](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9). Ainsi, vous obtiendrez l’alias d’envoyer et recevoir des messages.

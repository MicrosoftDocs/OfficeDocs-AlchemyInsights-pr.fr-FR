# <a name="add-or-remove-an-email-address-for-a-user"></a>Ajouter ou supprimer une adresse de messagerie pour un utilisateur

Pour commencer, [suivez ces étapes](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) dans le centre d’administration Microsoft 365.

 **Remarque**: vous pouvez avoir plusieurs adresses de messagerie pour un utilisateur, mais des *alias de messagerie* supplémentaires ne peuvent être utilisés que pour la réception de messages électroniques. Si vous devez envoyer un message électronique avec un alias, créez une [boîte aux lettres partagée](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox). De cette façon, vous obtiendrez l’alias permettant d’envoyer et de recevoir des messages.

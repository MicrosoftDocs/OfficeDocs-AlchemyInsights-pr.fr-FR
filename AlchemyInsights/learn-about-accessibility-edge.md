# <a name="learn-about-accessibility-in-microsoft-edge"></a>En savoir plus sur l’accessibilité dans Microsoft Edge

Dans Microsoft Edge, nous avons introduit l’[API UI Automation moderne (API UIA)](https://go.microsoft.com/fwlink/?linkid=2153423). Le passage à l'UIA est un investissement majeur dans l'accessibilité du navigateur, et il constitue la base d'une expérience web plus inclusive pour les utilisateurs qui dépendent de la technologie d'assistance dans Windows 10. 

Pour en savoir plus, consultez [Accessibilité dans Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2153512).

# <a name="need-to-create-an-email-signature"></a>Vous devez créer une signature électronique ?

Si vous avez un compte Outlook ou Outlook sur le compte web (Office 365 travail ou une école), vous pouvez créer une signature et ajouter automatiquement à tous les messages sortants ou sélectivement l’ajouter aux messages sortants spécifiques. Notez que votre signature électronique est différent dans Outlook et Outlook sur le web. Pour créer votre signature électronique, voir [créer et ajouter une signature aux messages dans Outlook](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) ou [créer et ajouter une signature électronique dans Outlook sur le site web](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx).

# <a name="solutions-for-unsupported-operating-system"></a>Solutions pour « Système d’exploitation non pris en charge »

Lorsque vous essayez d’installer Office 365/2016, vous pouvez voir le message d’erreur suivant si votre périphérique ne répond pas à la [configuration système requise](https://products.office.com/office-system-requirements).
  
 **« Système d’exploitation non pris en charge »**
  
Office 365/2016 nécessite :
  
PC : Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 ou Windows Server 2008 R2
  
OU
  
Mac OS X 10.10
  
Si ce message lorsque vous essayez d’installer Office sur un appareil mobile ou un Chromebook, voir [configurer les applications Office et de messagerie sur un appareil mobile](https://support.office.com/article/7dabb6cb-0046-40b6-81fe-767e0b1f014f?wt.mc_id=Alchemy_ClientDIA.aspx).

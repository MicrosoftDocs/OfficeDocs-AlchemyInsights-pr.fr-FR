# <a name="57750-client-blocked-from-sending-from-unregistered-domain"></a>Client 5.7.750 interdit d’envoi à partir d’un domaine non enregistré

L’erreur se produit lorsqu’un grand nombre de messages sont envoyés à partir de domaines qui ne sont pas configurés dans votre client (ajoutés en tant que domaines acceptés et validés).

Pour éviter cette erreur, vous pouvez utiliser un connecteur de flux de messagerie basé sur un certificat où le domaine du certificat est un domaine configuré, ou vous pouvez configurer tous les domaines d’envoi.

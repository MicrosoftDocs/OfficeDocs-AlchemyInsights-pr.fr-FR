# <a name="calendar-freebusy-issues-in-hybrid-environments"></a>Problèmes de disponibilité du calendrier dans les environnements hybrides

Si les utilisateurs rencontrent des problèmes de visualisation du calendrier et de disponibilité des informations dans les environnements hybrides, réexécutez l’Assistant Configuration hybride. Pour plus d’informations sur l’Assistant Configuration hybride, consultez [Assistant Configuration hybride](https://go.microsoft.com/fwlink/p/?linkid=528149).
  
Pour résoudre les problèmes comme suit dans les environnements hybrides, voir [Troubleshoot un déploiement hybride](https://technet.microsoft.com/library/jj659053.aspx).

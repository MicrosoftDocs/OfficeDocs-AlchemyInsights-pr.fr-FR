# <a name="550-57501-access-denied-spam-abuse-detected"></a>550 5.7.501 accès refusé, abus du courrier indésirable détecté

En règle générale, ce message s’affiche lorsque les utilisateurs envoient des messages électroniques provenant d’adresses IP à l’aide de la première *. onmicrosoft.com* domaine auquel est affecté aux nouveaux clients dans Office 365. Pour résoudre ce problème, la plus simple consiste à : 
  
1. [Ajouter un domaine à votre client](https://support.office.com/article/6383f56d-3d09-4dcb-9b41-b5f5a5efd611.aspx).
    
2. [Modifier l’adresse de messagerie principale de vos utilisateurs](https://support.office.com/article/fb5ac074-e203-4e1f-9843-b9d1a3e03297.aspx) vers le nouveau domaine personnalisé vous venez d’ajouter.

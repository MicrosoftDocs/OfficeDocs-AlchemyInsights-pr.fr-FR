# <a name="change-the-domain-for-office-365-group"></a>Modifier le domaine de groupe dans Office 365

Si vous souhaitez modifier le domaine d’un groupe d’Office 365 lors de la création, suivez les étapes de [Choisir le domaine à utiliser lors de la création de groupes d’Office 365](https://support.office.com/article/7cf5655d-e523-4bc3-a93b-3ccebf44a01a.aspx).
  
Une fois le groupe d’Office 365 est déjà créé, il n’est actuellement aucun possible de modifier le domaine. Le seul moyen consiste à créer un groupe Office 365 et supprimer l’ancien si nécessaire (les adresses de messagerie supplémentaires peuvent être ajoutés par le biais du [Centre d’administration Exchange](https://support.office.com/article/https://outlook.office365.com/ecp.aspx)).

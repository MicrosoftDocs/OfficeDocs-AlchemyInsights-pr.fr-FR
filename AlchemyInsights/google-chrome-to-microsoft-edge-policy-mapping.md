# <a name="learn-about-mapping-between-google-chrome-policies-and-microsoft-edge-policies"></a>En savoir plus sur le mappage entre les stratégies Google Chrome et les stratégies Microsoft Edge

Les stratégies Google Chrome sont mappées aux stratégies Microsoft Edge appropriées prises en charge dans la version 80 du navigateur. Pour en savoir plus, consultez la rubrique [Google Chrome to Microsoft Edge Policy Mapping](https://go.microsoft.com/fwlink/?linkid=2141933).

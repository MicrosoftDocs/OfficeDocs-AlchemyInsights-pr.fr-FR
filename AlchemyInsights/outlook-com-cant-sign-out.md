# <a name="unable-to-sign-out-of-outlookcom"></a>Impossible de se déconnecter de Outlook.com

Pour vous déconnecter de Outlook.com, sélectionnez votre nom ou votre image dans le coin supérieur droit de l’écran, puis sélectionnez **déconnexion**.

Si vous ne voyez pas votre image en haut de l’écran, vérifiez si le blocage des publicités est activé. Cela peut empêcher l’affichage de l’image.

Si vous ne parvenez toujours pas à vous déconnecter de Outlook.com, cliquez sur [https://login.live.com/logout.srf](https://login.live.com/logout.srf) fermer toutes les fenêtres de navigateur.

# <a name="profile-property-synchronization"></a>Propriété synchronisation des profils

Il n’est pas possible de mapper des propriétés supplémentaires pour l’importation Active Directory à synchroniser d’Azure Active Directory à l’Application de profil utilisateur. L’importation AD est limitée à un jeu de propriétés à garantir des performances cohérentes du travail du minuteur préconfiguré.
  
[Pour plus d’informations sur les propriétés de profil de synchronisation](https://go.microsoft.com/fwlink/?linkid=875671)

# <a name="access-denied-banned-sender"></a>Accès refusé, interdit l’expéditeur

 **Erreur**: erreur SMTP à partir d’un serveur distant pour RCPT TO command, hôte : *xxxx* -raison com.mail.protection.outlook.com ( *xxx.xxx.xxx.xxx* ) : 550 5.7.511 accès refusé, interdit l’expéditeur [ *xx.xxx.x.xxx* ]. 
  
Si vous rencontrez cette erreur lors de l’envoi de courrier électronique à un destinataire d’Office 365, supprimer le compte affecté à partir de la liste des expéditeurs bloqués en [https://sender.office.com](https://sender.office.com).

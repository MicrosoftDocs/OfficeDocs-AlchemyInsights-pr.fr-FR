# <a name="move-a-phone-number-from-another-provider-to-microsoft"></a>Déplacer un numéro de téléphone d’un autre fournisseur vers Microsoft

Avez-vous besoin de déplacer un numéro de téléphone d’un autre fournisseur vers Microsoft ? Vous pouvez trouver les numéros de téléphone de différents pays et régions dans [la zone gérer les numéros de téléphone de votre organisation](https://docs.microsoft.com/SkypeForBusiness/what-are-calling-plans-in-office-365/manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization). Choisissez votre pays ou région pour trouver des informations sur le transfert des numéros.

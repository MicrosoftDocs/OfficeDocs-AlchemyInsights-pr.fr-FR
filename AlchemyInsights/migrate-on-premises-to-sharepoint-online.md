# <a name="migrate-on-premises-to-sharepoint-online"></a>Migrer locaux vers SharePoint Online

Pour migrer à partir d’une batterie de serveurs SharePoint sur site vers SharePoint Online, vous pouvez utiliser l’outil de Migration de SharePoint. Pour plus d’informations sur l’outil, consultez la rubrique [migrer vers SharePoint Online.](https://go.microsoft.com/fwlink/?linkid=2019574)

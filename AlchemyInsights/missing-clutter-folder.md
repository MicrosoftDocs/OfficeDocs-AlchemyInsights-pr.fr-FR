# <a name="missing-clutter-folder"></a>Dossier pêle-mêle manquant

Si vous ne voyez pas de dossier Courrier pêle-mêle dans votre boîte aux lettres, vous devrez avoir l’option de Boîte de réception Prioritaire. Voir [Boîte de réception Prioritaire pour Outlook](https://support.office.com/article/focused-inbox-for-outlook-f445ad7f-02f4-4294-a82e-71d8964e3978) pour plus d’informations.

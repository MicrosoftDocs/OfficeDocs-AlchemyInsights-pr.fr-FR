# <a name="webpage-html-web-resources-limitations"></a>Limitations relatives aux ressources Web (HTML) pour les pages Web

* Une ressource Web HTML ne peut pas contenir de code qui doit être exécuté sur le serveur. Les pages ASP.NET ne peuvent pas être téléchargées en tant que ressources Web HTML.

* Les ressources Web HTML ne peuvent accepter qu’un nombre limité de paramètres de chaîne de requête. Pour plus d’informations, [repassez les paramètres aux ressources Web HTML](https://docs.microsoft.com/dynamics365/customer-engagement/developer/webpage-html-web-resources#BKMK_PassingParametersToWebResources) .

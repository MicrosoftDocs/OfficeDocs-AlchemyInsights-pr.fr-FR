# <a name="resolve-office-activation-error-0xd000000c"></a>Résoudre l’erreur d’activation Office 0xD000000C

- Vérifiez que votre connexion Internet fonctionne sur le Mac sur lequel vous effectuez l’installation. Pour ce faire, ouvrez votre navigateur Internet, puis rendez-vous sur www.microsoft.com. Si la page charge, votre connexion Internet fonctionne.

- Quittez toutes les applications Office.

- Téléchargez et exécutez l’[outil de suppression de licence](https://go.microsoft.com/fwlink/?linkid=849815) pour supprimer toutes les entrées du trousseau Office.

- Redémarrez une application Office et suivez les invites d’activation.

Si vous souhaitez en savoir plus, consultez :

- [Erreur 0xD000000C lors de l’activation d’Office pour Mac](https://support.office.com/article/error-0xd000000c-when-activating-office-for-mac-da865931-4658-4829-ba2d-8133390c6d25)

- [Que faire si vous ne pouvez pas installer ou activer Office pour Mac ?](https://support.office.com/article/what-to-try-if-you-can-t-install-or-activate-office-for-mac-5efba2b4-b1e6-4e5f-bf3c-6ab945d03dea)

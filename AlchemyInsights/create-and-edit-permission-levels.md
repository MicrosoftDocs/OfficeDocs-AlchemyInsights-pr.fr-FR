# <a name="working-with-sharepoint-permission-levels"></a>Utilisation des niveaux d’autorisation SharePoint

SharePoint est fourni avec des niveaux d’autorisation par défaut que vous pouvez utiliser, comme **Contribute** et **affichage uniquement**. Toutefois, si l’un de ces éléments ne répond pas à vos besoins, vous pouvez créer un nouveau niveau d’autorisation. N’oubliez pas qu’il est plus facile de suivre les niveaux d’autorisation si vous ne modifiez pas les paramètres par défaut.

[Créer et modifier des niveaux d’autorisation](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)

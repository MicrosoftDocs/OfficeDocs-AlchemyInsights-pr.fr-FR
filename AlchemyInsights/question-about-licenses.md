# <a name="question-about-licenses"></a>Question sur les licences

Si vous souhaitez connaître le nombre de licences que vous avez acheté, ou le nombre de licences est affecté à partir d’un abonnement :
  
1. Dans le Centre d'administration, accédez à la page [Abonnements](https://go.microsoft.com/fwlink/p/?linkid=842054), ou choisissez **Facturation** \> **Abonnements**.
    
2. Choisissez l’abonnement que vous souhaitez en savoir plus.
    
3. Sur le côté gauche de la carte d’abonnement est plus d’informations sur vos licences :
    
  - Le nombre de **acheté** est le nombre de licences que vous payez. 
    
  - Le numéro **attribué** est le nombre de licences attribuées à un utilisateur. 
    
4. Si vous souhaitez ajouter davantage de licences, ou si vous avez trop nombreuses, choisissez **Ajouter/supprimer des licences** pour régler le volume acheté.

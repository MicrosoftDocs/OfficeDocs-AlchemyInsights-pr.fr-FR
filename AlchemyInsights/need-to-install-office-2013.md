# <a name="office-2013-is-no-longer-supported-in-microsoft-365-subscriptions"></a>Office 2013 n’est plus pris en charge dans les abonnements Microsoft 365

Depuis le 28 février 2017, Office 2013 n’est plus pris en charge dans les abonnements Microsoft 365.
  
Pour garantir la sécurité de votre suite Office et obtenir toutes les fonctionnalités les plus récentes, installez gratuitement la dernière version d’Office dans le cadre de votre abonnement Microsoft 365. Consultez [la rubrique mettre à niveau vos utilisateurs vers le client Office le plus récent](https://docs.microsoft.com/microsoft-365/admin/setup/upgrade-users-to-latest-office-client).

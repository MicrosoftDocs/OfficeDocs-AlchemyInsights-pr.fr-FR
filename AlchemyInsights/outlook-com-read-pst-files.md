# <a name="pst-data-files-in-outlookcom"></a>Fichiers de données PST dans Outlook.com

Il n’est pas possible d’importer une Outlook. PST sur votre compte Outlook.com à l’aide d’un navigateur.

Pour importer un. Fichier PST, vous devez utiliser l’application de bureau Outlook. Pour en savoir plus, consultez [importer du courrier, des contacts et un calendrier à partir d’un fichier. pst Outlook.](https://support.office.com/article/431a8e9a-f99f-4d5f-ae48-ded54b3440ac?wt.mc_id=Office_Outlook_com_Alchemy)

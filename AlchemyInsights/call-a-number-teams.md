# <a name="call-a-phone-number-in-teams"></a>Appeler un numéro de téléphone dans teams

Accédez à  **appels**, sélectionnez  **composer un numéro**, puis utilisez le pavé de numérotation pour entrer le numéro. Ensuite, cliquez sur  **appeler**.

Si vous ne voyez pas l’option permettant de composer un numéro, il se peut que votre organisation n’ait pas acheté un forfait d’appels. Toutefois, vous pouvez toujours passer des appels à d’autres personnes dans Teams.  

Pour plus d’informations, consultez [la rubrique appeler un numéro de téléphone à partir de teams](https://support.microsoft.com/office/20d24ace-2851-4c29-8441-30dd2a5cf078).

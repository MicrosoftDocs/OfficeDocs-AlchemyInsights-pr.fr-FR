# <a name="sharepoint-online-calendar-overlay"></a>Superposition de calendriers SharePoint Online

Vous pouvez superposer un autre calendrier SharePoint à condition que les deux calendriers SharePoint se trouvent dans la même collection de sites. La possibilité d’utiliser la superposition de calendrier dans un calendrier SharePoint Online ou SharePoint Server qui est extraite d’Exchange Online n’est pas prise en charge.

SharePoint Online offre une expérience moderne avec les calendriers de groupe Microsoft 365. Pour plus d’informations, consultez la rubrique [Utiliser le composant WebPart Calendrier de groupe](https://support.microsoft.com/fr-FR/office/use-the-group-calendar-web-part-eaf3c04d-5699-48cb-8b5e-3caa887d51ce).

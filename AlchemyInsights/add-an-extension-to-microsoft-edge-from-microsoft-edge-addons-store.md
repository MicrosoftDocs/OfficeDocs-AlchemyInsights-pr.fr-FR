# <a name="add-an-extension-to-microsoft-edge-from-the-microsoft-edge-add-ons-store"></a>Ajouter une extension à Microsoft Edge à partir du magasin de compléments Microsoft Edge

Pour ajouter une extension à Microsoft Edge à partir du magasin de compléments Microsoft Edge :

- Ouvrez Microsoft Edge.
- Sélectionnez **les extensions... > > [obtenir des extensions pour Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2136408)**.
- Sélectionnez l’extension que vous souhaitez ajouter, puis sélectionnez **obtenir**.
- Dans la boîte de dialogue de confirmation, vérifiez attentivement les modifications que l’extension peut apporter, puis sélectionnez **Ajouter une extension**.

Vous recevrez ensuite une confirmation de l’ajout de l’extension.

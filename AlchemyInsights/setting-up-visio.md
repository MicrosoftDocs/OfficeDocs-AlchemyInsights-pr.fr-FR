# <a name="setting-up-visio"></a>Configuration de Visio

Il semble que vous avez besoin d’aide pour configurer Visio avec votre abonnement Microsoft 365.
  
Avis pour les utilisateurs Mac : le produit Visio n’est pas disponible pour MacOS pour le moment. Si vous disposez d’un PC, vous devriez pouvoir installer le logiciel en suivant les étapes ci-dessous.
  
Vous pouvez suivre la procédure décrite dans [cet article](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710.aspx) pour terminer votre configuration.

# <a name="setting-up-visio"></a>Configuration de Visio

Semble que vous avez besoin d’aide Visio avec votre abonnement à Office 365.
  
Notification pour les utilisateurs Mac : produit Visio n’est pas disponible pour Mac OS à ce stade. Si vous avez un PC vous devez être en mesure d’installer le logiciel à l’aide de la procédure ci-dessous.
  
Vous pouvez suivre la procédure décrite dans [cet article](https://support.office.com/article/https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710.aspx) pour terminer votre configuration.

# <a name="long-running-script"></a>Script long en cours d'exécution

Nous avons connaissance des problèmes liés au script «longue exécution» dans Outlook.com. Pour résoudre ce problème, utilisez la version bêta d'Outlook.com.
  
[En savoir plus sur la version bêta d'Outlook.com.](https://go.microsoft.com/fwlink/p/?linkid=874356)

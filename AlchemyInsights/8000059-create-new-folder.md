# <a name="create-a-new-folder"></a>Créer un nouveau dossier

1. Les **dossiers** dans le volet de navigation, cliquez sur **Nouveau dossier**. 
    
2. Tapez un nom pour le dossier et appuyez sur ENTRÉE.
    
Pour plus d’informations, voir [créer un dossier dans Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a).

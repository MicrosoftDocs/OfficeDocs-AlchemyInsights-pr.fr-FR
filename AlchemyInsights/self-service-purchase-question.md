# <a name="self-service-purchase"></a>Achat en libre-service

**Quelles sont les fonctionnalités dont dispose un administrateur pour les achats en libre-service ?**

Les administrateurs peuvent afficher tous les achats en libre-service effectués dans leur organisation dans le centre d’administration Microsoft 365. Ils peuvent voir le produit, le nom de l’acheteur, les abonnements achetés, la date d’expiration, l’historique des commandes, le prix d’achat et les utilisateurs affectés pour chaque achat en libre-service.  Si cela est nécessaire pour leur organisation, les administrateurs pourront désactiver les achats en libre-service sur une base par produit via PowerShell.  Les administrateurs ont les mêmes stratégies de gestion des données et d’accès que les produits achetés via l’achat en libre-service ou de manière centralisée.

# <a name="setting-up-project"></a>Configuration du projet

Il semble que vous ayez besoin d’aide pour configurer Project avec votre abonnement Office 365.
Notification pour les utilisateurs Mac: le produit du projet n’est pas disponible pour MacOS pour le moment. Si vous disposez d’un PC, vous devriez pouvoir installer le logiciel en suivant les étapes ci-dessous.
  
Vous pouvez suivre les étapes décrites dans [cet article](https://support.office.com/article/7059249b-d9fe-4d61-ab96-5c5bf435f281.aspx)pour terminer votre configuration.

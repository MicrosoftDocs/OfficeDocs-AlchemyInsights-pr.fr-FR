# <a name="how-do-i-turn-off-skype-in-outlookcom"></a>Comment désactiver Skype dans Outlook.com?

Vous ne pouvez pas supprimer Skype de Outlook.com, mais vous pouvez désactiver les notifications:

1. Ouvrez les notifications**générales** > **[des](https://outlook.live.com/mail/options/general/notifications)** **paramètres** > . 
2. Sous **conversation et appel**, désactivez les types de notification dont vous n’avez plus besoin.

Pour plus d’informations sur l’utilisation de Skype, consultez la discussion sur la [famille et les amis sur Skype dans Outlook.com](https://support.office.com/article/83c6a5b1-3921-479c-b9e9-e753ce59c1fa).

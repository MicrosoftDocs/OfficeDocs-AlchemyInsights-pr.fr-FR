# <a name="use-whiteboard-with-microsoft-365"></a>Utiliser le Tableau blanc collaboratif avec Microsoft 365

Le Tableau blanc collaboratif Microsoft est une surface de dessin libre numérique sur laquelle se réunissent personnes, contenu et idées. 

- [Activez le Tableau blanc collaboratif](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920#bkmk_07) en tant qu’administrateur Microsoft 365. 

- Utilisez le [Tableau blanc collaboratif avec Microsoft Teams](https://support.microsoft.com/fr-FR/office/use-whiteboard-in-microsoft-teams-7a6e7218-e9dc-4ccc-89aa-b1a0bb9c31ee). 

- [Collaborez](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920#bkmk_27) avec une autre personne sur le Tableau blanc collaboratif. 

Pour plus d’informations, veuillez consulter l’[aide de Tableau blanc collaboratif Microsoft](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920).

# <a name="restoring-a-deleted-team-site"></a>Restauration d’un Site d’équipe supprimé

Avez-vous, en tant que propriétaire de l’équipe, accidentellement supprimé une équipe ? Il se produit, mais vous pouvez toujours récupérer.
  
Pour restaurer une équipe supprimé pour Microsoft Teams, suivez les étapes décrites ici : [restauration d’une équipe supprimé](https://blogs.technet.microsoft.com/skypehybridguy/2017/07/23/restoring-a-deleted-team-in-microsoft-teams/)

# <a name="restoring-a-deleted-team-site"></a>Restauration d'un site d'équipe supprimé

Avez-vous supprimé accidentellement une équipe? Si elle a été inférieure à 30 jours depuis que vous avez supprimé l'équipe, vous avez la chance; vous pouvez le restaurer. Pour ce faire, suivez les étapes ci-dessous: [restauration d'une équipe supprimée](https://blogs.technet.microsoft.com/skypehybridguy/2017/07/23/restoring-a-deleted-team-in-microsoft-teams/)

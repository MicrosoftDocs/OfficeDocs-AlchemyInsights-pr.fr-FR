# <a name="problems-sharing-with-outlook-customer-manager"></a>Problèmes de partage avec le gestionnaire de clients Outlook

Si vous rencontrez des problèmes de partage avec le gestionnaire de clients Outlook, nous vous recommandons de commencer par effacer le cache de votre navigateur. Par exemple dans Internet Explorer, vous devez :

1. Accédez à **paramètres** > **généraux** > paramètres sous **Browsing history** > **les bases de données et les caches**d’historique de navigation.
2. Supprimez les fichiers de cache pour le site Web « outlookapps.com ».
3. Redémarrez Outlook.

Pour plus d’informations sur le partage, voir [partager des informations dans le gestionnaire de clients Outlook](https://support.office.com/article/4f26cc69-67da-4cd5-b344-02d1a4799310%20).

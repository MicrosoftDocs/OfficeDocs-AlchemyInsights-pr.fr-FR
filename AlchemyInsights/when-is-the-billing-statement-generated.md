# <a name="when-is-the-billing-statement-generated"></a>Lorsque l’instruction facturation est générée

Chaque mois, vous recevrez un courrier électronique qui vous indique que votre nouvelle instruction facturation est disponible dans le centre d’administration d’Office 365. Votre facture contient deux pages. Page 1 est la facture de synthèse et contient des informations générales sur la facture, la commande, le montant d’échéance, comment effectuer un paiement et comment contacter le support technique. Page 2 contient plus d’informations sur l’activité de facturation pour chaque abonnement.
  
Pour en savoir plus sur les concepts de facturation, voir [comprendre la facture](https://support.office.com/article/0724b428-fb59-4962-8c37-6674166d7507).

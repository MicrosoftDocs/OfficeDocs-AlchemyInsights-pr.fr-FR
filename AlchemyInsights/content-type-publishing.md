# <a name="content-type-publishing"></a>Publication de type de contenu

Si vous souhaitez que les types de contenu soient disponibles dans l’ensemble de votre organisation pour tous les utilisateurs, il est recommandé de publier dans toutes les collections de sites. Pour publier un type de contenu, voir [publier un type de contenu à partir d’un concentrateur de publication de contenu](https://support.office.com/article/publish-a-content-type-from-a-content-publishing-hub-58081155-118d-4e7a-9cc5-d43b5dbb7d02).

# <a name="change-the-domain-for-office-365-group"></a>Modifier le domaine de groupe dans Office 365

Si vous souhaitez modifier le domaine d’un groupe d’Office 365 lors de la création, suivez les étapes de [Choisir le domaine à utiliser lors de la création de groupes d’Office 365](https://docs.microsoft.com/office365/admin/create-groups/choose-domain-to-create-groups).

Une fois le groupe d’Office 365 est déjà créé, il n’est actuellement aucun possible de modifier le domaine. Le seul moyen consiste à créer un groupe Office 365 et supprimer l’ancien si nécessaire (les adresses de messagerie supplémentaires peuvent être ajoutés par le biais du [Centre d’administration Exchange](https://outlook.office365.com/ecp).

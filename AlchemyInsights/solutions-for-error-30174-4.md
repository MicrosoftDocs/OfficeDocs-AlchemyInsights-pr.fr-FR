# <a name="error-code-30174-4-when-installing-office"></a>Code d'erreur 30174-4 lors de l'installation d'Office

Cette erreur peut apparaître si votre logiciel antivirus, votre pare-feu, vos paramètres de proxy ou votre connexion vous empêche d'installer Office. Here are some things you can try to fix the problem.
  
- Utiliser une connexion câblée
    
- Désactiver temporairement les paramètres de proxy
    
- Désactiver temporairement le logiciel antivirus

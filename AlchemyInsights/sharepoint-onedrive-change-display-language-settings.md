# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>Modifier les paramètres de langue d’affichage dans SharePoint ou OneDrive 

Vous ne pouvez pas modifier les paramètres de langue par défaut pour SharePoint ou OneDrive, car ces paramètres ont été définis lors de la création de votre abonnement Office 365. Toutefois, vous pouvez modifier vos paramètres de langue d’affichage.

Pour modifier vos paramètres personnels, reportez-vous à [la rubrique modifier vos paramètres de langue et de région](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7).

Pour modifier l’interface utilisateur du site, reportez-vous à [la rubrique choisir les langues que vous souhaitez mettre à disposition pour l’interface utilisateur d’un site](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8).

Pour plus d’informations sur la résolution des problèmes liés aux paramètres de langue, voir [vous ne pouvez pas modifier la langue par défaut d’une collection de sites SharePoint Online](https://support.office.com/article/you-can-t-change-the-default-language-for-a-sharepoint-online-site-collection-40dda07e-6b41-49e9-9828-8805dcb92964).

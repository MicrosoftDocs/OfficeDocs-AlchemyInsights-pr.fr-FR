# <a name="inbox-not-loading-with-firefox"></a>Boîte de réception non chargée avec Firefox

Certains modules complémentaires dans le navigateur Firefox peuvent empêcher le chargement de votre boîte de réception Outlook.com.
  
Pour savoir comment [désactiver ou supprimer des modules](https://support.mozilla.org/kb/disable-or-remove-add-ons)complémentaires, consultez la prise en charge de Firefox.

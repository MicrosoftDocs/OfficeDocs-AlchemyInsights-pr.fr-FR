# <a name="how-to-cancel-software-as-a-service-apps"></a>Procédure d’annulation des applications logicielles en tant que service

1. Dans le centre d’administration, sur la page **facturation**de  >  [vos produits](https://go.microsoft.com/fwlink/p/?linkid=842054) .
2. Sur la page **produits** , faites défiler jusqu’à la section **applications** , puis recherchez l’abonnement que vous souhaitez annuler. 
3. Sélectionnez **autres actions** (trois points), puis **Annuler l’abonnement**.
4. Dans le volet **Annuler l’abonnement** , choisissez la raison pour laquelle vous annulez l’abonnement. Vous pouvez également fournir des commentaires.
5. Sélectionnez **Enregistrer**.

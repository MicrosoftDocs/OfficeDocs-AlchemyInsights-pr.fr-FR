# <a name="troubleshoot-error-code-aadsts50011"></a>Résoudre des problèmes de code d’erreur AADSTS50011

Pour résoudre l’erreur AADSTS50011, effectuez l’étape recommandée décrite ci-dessous.

**AADSTS50011** : InvalidReplyTo - L’adresse de réponse est manquante, incorrectement configurée ou ne correspond pas aux adresses de réponse configurées pour l’application.

Pour résoudre ce problème, veillez à ajouter cette adresse de réponse manquante à l'application Azure Active Directory (AD) ou demandez à une personne ayant les autorisations nécessaires pour gérer votre application dans AD de le faire pour vous. Pour plus d’informations, consultez l’article de résolution des problèmes pour l’erreur [AADSTS50011](https://docs.microsoft.com/troubleshoot/azure/active-directory/error-code-aadsts50011-reply-url-mismatch).

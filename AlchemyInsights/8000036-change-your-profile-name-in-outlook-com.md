# <a name="change-your-profile-name-in-outlookcom"></a>Modifier le nom de votre profil dans Outlook.com

1. Accédez à [votre Microsoft les informations de compte](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. Sélectionnez **Modifier le nom**. 
    
3. Entrez le nom et le nom de famille, puis sélectionnez **Enregistrer**. 
    
Patientez jusqu'à 24 heures pour que les modifications à votre photo dans votre profil ou un nom à en vigueur.

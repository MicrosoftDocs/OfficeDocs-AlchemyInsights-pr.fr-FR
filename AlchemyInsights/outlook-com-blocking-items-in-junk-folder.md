# <a name="blocking-items-in-your-junk-email-folder"></a>Blocage des éléments dans votre dossier courrier indésirable

Le blocage d’un élément déjà présent dans le dossier courrier indésirable ne le supprime pas. Les éléments de votre [dossier courrier indésirable](https://outlook.live.com/mail/junkemail) sont automatiquement supprimés après 30 jours. Sélectionnez un élément dans votre dossier courrier indésirable pour savoir quand il sera supprimé.

En savoir plus sur la façon de [bloquer les expéditeurs ou de marquer le courrier comme courrier indésirable](https://support.office.com/article/a3ece97b-82f8-4a5e-9ac3-e92fa6427ae4).

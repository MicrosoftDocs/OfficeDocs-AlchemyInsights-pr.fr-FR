# <a name="create-users-in-hybrid-deployments"></a>Créer des utilisateurs dans des déploiements hybrides

Lorsque votre organisation est un déploiement hybride entre un environnement local et Office 365, vous devez créer tous les objets utilisateur dans votre annuaire Active Directory local. Vous créez les objets utilisateur dans l’unité d’organisation appropriée afin qu’ils soient synchronisés avec votre organisation Office 365.

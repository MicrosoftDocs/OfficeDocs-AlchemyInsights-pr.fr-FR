# <a name="restrict-default-video-upload-permissions"></a>Restreindre les autorisations de chargement vidéo par défaut

Lorsque les utilisateurs téléchargent une vidéo vers Microsoft Stream via le portail de flux en dehors du contexte d’un canal de groupe ou de groupe, la case à cocher « Autoriser tous les utilisateurs de l’entreprise à visualiser cette vidéo » est activée par défaut. Cela rend la vidéo disponible pour être visualisée par tous les membres de votre entreprise.

- [Définir les autorisations de chargement de la vidéo par défaut](https://docs.microsoft.com/stream/default-video-permissions)

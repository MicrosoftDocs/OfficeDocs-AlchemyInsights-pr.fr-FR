# <a name="troubleshooting-refresh-scenarios---tile-errors"></a>Dépannage des scénarios d’actualisation-erreurs de mosaïque

Veuillez patienter environ 10-15 minutes pour que l’actualisation soit reflétée dans les vignettes du tableau de bord. Si elle ne s’affiche toujours pas, réépinglez la visualisation sur le tableau de bord.

Pour plus d’informations, procédez comme suit:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)

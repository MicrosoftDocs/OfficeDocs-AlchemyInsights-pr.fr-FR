# <a name="email-forwarding-report"></a>Rapport concernant le transfert du courrier électronique

Pour afficher un rapport sur le nombre de messages transférés automatiquement au sein de votre organisation, consultez la rubrique [Informations sur le flux de courrier](https://docs.microsoft.com//office365/securitycompliance/mail-flow-insights-v2) dans le [Centre de conformité&amp; de la Sécurité Office 365](https://protection.office.com/#/homepage).
  
Remarque : si vous vous connectez pour la première fois au Centre de conformité &amp;de la sécurité, vous devrez peut-être actualiser le navigateur après quelques minutes pour voir toutes les options disponibles.

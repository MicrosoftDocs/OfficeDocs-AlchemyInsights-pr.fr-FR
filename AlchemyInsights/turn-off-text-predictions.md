# <a name="turn-off-text-predictions"></a>Désactiver les prédictions de texte

1. Sélectionnez **paramètres > afficher tous les paramètres Outlook > courrier > composition et répondre**.
2. Sous **prédictions de texte**, désélectionnez **suggérer des mots d’expressions comme je tape**.

Pour plus d’informations, consultez [la rubrique utiliser une technologie intelligente dans Outlook sur le Web et Outlook.com](https://support.microsoft.com/office/use-intelligent-technology-in-outlook-on-the-web-and-outlook-com-24b30683-8340-4b69-b8ac-4193ec528a70).

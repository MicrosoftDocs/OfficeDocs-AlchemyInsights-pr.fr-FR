# <a name="how-to-change-your-theme"></a>Comment modifier votre thème

Pour modifier votre thème dans Outlook.com ou Outlook sur le Web, accédez à **paramètres** et choisissez un thème sous **thème**. Pour afficher tous les thèmes disponibles, sélectionnez **Afficher tout** sous **thème**.

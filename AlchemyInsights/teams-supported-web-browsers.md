# <a name="teams-supported-web-browsers"></a>Navigateurs Web pris en charge par teams

Teams prend entièrement en charge les navigateurs Internet répertoriés dans l’article, [Limits and Specifications for Microsoft teams](https://docs.microsoft.com/en-us/microsoftteams/limits-specifications-teams#browsers), avec les exceptions indiquées pour les appels et les réunions.

# <a name="how-to-right-click-on-a-tablet"></a>Comment cliquer avec le bouton droit sur une tablette

Pour cliquer avec le bouton droit sur une surface ou une tablette Windows, appuyez et maintenez votre sélection pendant quelques secondes. Lorsque vous relâchez, le menu contextuel s’affiche.

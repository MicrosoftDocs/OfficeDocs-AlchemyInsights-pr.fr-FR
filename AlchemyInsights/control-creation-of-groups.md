# <a name="control-creation-of-microsoft-365-groups"></a>Contrôler la création de groupes Microsoft 365

Par défaut, tous les utilisateurs au sein de l’entreprise peuvent créer des groupes. Si nécessaire, vous pouvez modifier ce paramètre et attribuer le droit de créer des groupes à un ensemble de personnes au sein de votre entreprise. Notez que cela aura un impact sur la création de groupes à partir de n’importe quel service Microsoft 365 (par exemple, Outlook, teams, Planner, etc.).
  
Instructions : [gérer les personnes autorisées à créer des groupes Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)

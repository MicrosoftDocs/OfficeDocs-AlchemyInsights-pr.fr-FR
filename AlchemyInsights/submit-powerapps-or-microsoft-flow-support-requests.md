# <a name="submit-powerapps-or-microsoft-flow-support-requests"></a>Soumettre des demandes de support PowerApp ou Microsoft Flow

Si vous êtes un client **PowerApp** ou **Microsoft Flow** , envoyez votre ticket via le nouveau [Centre d’administration Power Platform](https://admin.powerplatform.microsoft.com/support?newTicket&product=15819). Le nouveau portail fournit un chemin plus direct vers le support assisté qui est optimisé pour les PowerApp et les autres produits Power Platform.

[Soumettre une demande de support de flux Microsoft](https://admin.powerplatform.microsoft.com/support?newTicket&product=Flow)

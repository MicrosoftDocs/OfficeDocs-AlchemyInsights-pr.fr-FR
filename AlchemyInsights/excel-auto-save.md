# <a name="what-is-autosave"></a>Qu’est-ce que l’enregistrement automatique ?

Autosave est une fonctionnalité disponible dans Excel, Word et PowerPoint pour les abonnés Microsoft 365 qui enregistrent votre fichier automatiquement, toutes les quelques secondes, au fur et à mesure que vous travaillez. 

Pour consulter les questions fréquemment posées, consultez [qu’est-ce que l’enregistrement automatique ?](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)

# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Version prise en charge de l’outil de Migration de SharePoint



L’outil de Migration de SharePoint seulement prend en charge des migrations à partir de SharePoint 2013. Versions 2010 et 2016 ne sont pas pris en charge pour l’instant.
  
Pour plus d’informations, voir [Présentation de l’outil de Migration de SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).

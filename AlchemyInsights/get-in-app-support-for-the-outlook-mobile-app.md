# <a name="get-in-app-support-for-the-outlook-mobile-app"></a>Obtenir la prise en charge de l’application mobile Outlook pour l’application

1. Dans l’application mobile Outlook, ouvrez le **menu**.

2. Sélectionnez le point d’interrogation pour ouvrir l’écran de commentaires de l' **aide &amp; ** .

3. Appuyez sur **contacter le support technique** et expliquez votre problème.

4. Appuyez sur **Envoyer** et vous recevrez une notification d’application lorsqu’un agent du support technique vous contactera.

Pour obtenir des instructions détaillées, voir [obtenir de l’aide dans l’application pour Outlook pour iOS et Android](https://support.office.com/article/218a22d1-9fa5-4889-b689-de1c63493243.aspx#ID0EAABAAA=Contact_Support).

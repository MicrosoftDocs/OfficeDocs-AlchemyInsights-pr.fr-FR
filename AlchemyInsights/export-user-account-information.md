# <a name="export-user-account-information"></a>Exporter les informations de compte d'utilisateur

Pour exporter des informations de compte d'utilisateur dans le Centre d'administration Microsoft 365, notamment des informations sur les licences, la stratégie de mot de passe, les comptes bloqués, l'état de synchronisation AAD, etc., allez sur la page Utilisateurs actifs, puis sélectionnez Exporter les  >  [](https://go.microsoft.com/fwlink/p/?linkid=834822) utilisateurs.

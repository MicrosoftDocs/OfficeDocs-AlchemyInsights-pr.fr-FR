2. <span data-ttu-id="1cf21-p102">**Les utilisateurs externes dans un groupe externe** - groupes externes autoriser les utilisateurs externes. L’étendue de l’accès d’un utilisateur externe est limitée au groupe externe. 3. <span data-ttu-id="1cf21-p103">**Les utilisateurs externes d’une conversation Yammer dans un groupe interne** - cette option autorise les participants externes à ajouter à une conversation particulière. 4. <span data-ttu-id="1cf21-p104">**Réseaux externes** - cette option vous permet d’inviter des utilisateurs externes à un réseau externe et vérifié que les administrateurs peuvent créer dans votre réseau.

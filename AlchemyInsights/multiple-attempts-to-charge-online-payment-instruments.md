# <a name="multiple-attempts-to-charge-online-payment-instruments"></a>Plusieurs tentatives de chargement des instruments de paiement en ligne

Nous effectuons plusieurs tentatives de facturation des instruments de paiement en ligne avant d’entreprendre toute action sur le service. Une fois que vous avez sélectionné un mode de paiement, tous les montants échus doivent être payés avec le mode de paiement sélectionné. Pour mettre à jour les informations de paiement actuelles pour votre abonnement, suivez les étapes de la procédure [Ajouter, mettre à jour ou supprimer une carte bancaire ou un compte bancaire dans Office 365 pour les entreprises](https://support.office.com/article/30ba9c83-50d8-4020-90ed-830a5b8c8724).

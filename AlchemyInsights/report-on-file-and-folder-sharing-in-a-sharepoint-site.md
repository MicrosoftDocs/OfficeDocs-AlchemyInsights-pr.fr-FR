# <a name="report-on-file-and-folder-sharing-in-a-sharepoint-site"></a>Rapport sur le partage de fichiers et de dossiers dans un site SharePoint

Vous pouvez créer un fichier CSV de chaque fichier, utilisateur, autorisation et lien uniques sur un site SharePoint donné ou une bibliothèque OneDrive. Cela peut vous aider à comprendre l’utilisation du partage et la possibilité de partager des fichiers ou dossiers avec des invités. Pour plus d’informations, consultez [Rapport sur le partage de fichiers et de dossiers dans un site SharePoint](https://docs.microsoft.com/sharepoint/sharing-reports).

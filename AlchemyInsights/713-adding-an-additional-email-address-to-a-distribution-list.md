# <a name="add-an-email-address-for-a-distribution-group"></a>Ajouter une adresse de messagerie pour un groupe de distribution

Pour ajouter des adresses de messagerie pour un groupe de distribution existant, procédez comme suit :
  
1. Dans le Centre d'administration Exchange, accédez à [Destinataires](https://outlook.office365.com/ecp/) **** \>.
    
2. Sélectionnez le groupe de distribution que vous souhaitez modifier, puis cliquez sur **Modifier**.
    
3. Dans les propriétés du groupe, accédez à l’onglet **options de messagerie** pour ajouter des adresses de messagerie. 
    
Pour plus d’informations, voir [options de messagerie](https://technet.microsoft.com/library/bb124513.aspx#emailoptions).

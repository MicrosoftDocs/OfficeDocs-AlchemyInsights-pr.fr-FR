# <a name="change-your-profile-name"></a>Modifier le nom de votre profil

1. Accédez à [vos informations de compte Microsoft](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. Sélectionnez **modifier le nom**. 
    
3. Entrez un prénom et un nom, puis sélectionnez **Enregistrer**. 
    
Veuillez patienter jusqu'à 24 heures pour que les modifications apportées à votre nom de profil prennent effet.
  
Pour en savoir plus, consultez [la rubrique modifier votre photo ou le nom de votre profil dans Outlook.com](https://go.microsoft.com/fwlink/?linkid=873110).

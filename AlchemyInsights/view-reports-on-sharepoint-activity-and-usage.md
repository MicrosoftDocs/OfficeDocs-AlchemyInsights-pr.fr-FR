# <a name="view-reports-on-sharepoint-activity-and-usage"></a>Afficher des rapports sur l’activité et l’utilisation de SharePoint

Go to the [Usage reports page in the Microsoft 365 admin center](https://admin.microsoft.com/AdminPortal/Home). Sous **Sélectionner un rapport,** choisissez le rapport d’activité ou d’utilisation SharePoint.

Pour plus d’informations, voir [à propos des rapports SharePoint.](https://go.microsoft.com/fwlink/?linkid=875240)

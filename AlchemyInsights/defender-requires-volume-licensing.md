# <a name="microsoft-defender-for-endpoint-requires-microsoft-volume-licensing"></a>Microsoft Defender pour le point de terminaison requiert une licence en volume Microsoft

Microsoft Defender pour le point de terminaison nécessite l’une des offres de licence en volume Microsoft suivantes :

- Windows 10 Entreprise E5
- Windows 10 Éducation A5
- Microsoft 365 E5 (M365 E5) qui inclut Windows 10 Entreprise E5
- Microsoft 365 A5 (M365 A5)
- Microsoft 365 E5 Sécurité
- Sécurité Microsoft 365 A5
- Microsoft Defender pour point de terminaison

Pour plus d’informations, voir [Minimum requirements for Microsoft Defender for Endpoint](https://docs.microsoft.com/microsoft-365/security/defender-endpoint/minimum-requirements).

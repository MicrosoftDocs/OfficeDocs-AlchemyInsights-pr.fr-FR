# <a name="view-your-users-microsoft-365-apps-activity"></a>Afficher l’activité des applications Microsoft 365 de vos utilisateurs

Le [rapport Activation d'Office](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft-office-activations?view=o365-worldwide) vous offre un aperçu des utilisateurs ayant activé leur abonnement Microsoft 365 sur au moins un appareil.

De plus, vous pouvez consulter l’activité des applications Microsoft 365 dans les applications Microsoft à l’aide du [Rapport sur l’utilisation des applications Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft365-apps-usage?view=o365-worldwide).

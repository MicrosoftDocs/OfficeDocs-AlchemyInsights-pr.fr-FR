# <a name="remove-a-password-from-a-word-document"></a>Supprimer un mot de passe d’un document Word

Tant que vous connaissiez le mot de passe d’origine, la suppression d’un mot de passe d’un document est un processus simple.

1. Ouvrez le document et entrez le mot de passe.
2. Accédez à infos sur le **fichier**  >  **Info**  >  **protéger le document**  >  **avec le mot de passe**.
3. Effacez le mot de passe dans la **zone mot de passe**, puis sélectionnez **OK**.

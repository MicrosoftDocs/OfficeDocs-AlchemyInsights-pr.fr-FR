# <a name="change-your-pin"></a>Modifier votre code confidentiel

Outlook.com n’utilise pas de code confidentiel, mais vous pouvez vous connecter à votre PC avec un code confidentiel ou Windows Hello. Pour modifier votre code confidentiel dans Windows 10:

1. Sélectionnez le bouton Démarrer ou appuyez sur la touche Windows de votre clavier, puis sélectionnez**options de connexion**des**comptes** > de **paramètres** > .
2. Sélectionnez le bouton **modifier** sous **code confidentiel**.
3. Entrez votre code confidentiel actuel, puis saisissez et confirmez votre nouveau code confidentiel sous.

[En savoir plus sur Windows Hello.](https://support.microsoft.com/help/17215/)

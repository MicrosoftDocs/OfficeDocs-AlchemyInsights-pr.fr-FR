# <a name="put-apps-on-the-desktop"></a>Placer les applications sur le Bureau

Recherchez l’application dans le menu **Démarrer** , puis saisissez l’icône de l’application en appuyant sur le bouton gauche de la souris et en dépositionnant l’icône sur le bureau.

Vous pouvez également appuyer sur la **touche Windows + R** , puis taper la commande suivante :

`shell:appsfolder`

Cette opération ouvre le dossier **applications** dans l' **Explorateur de fichiers**. Vous pouvez maintenant faire glisser et déplacer l’icône de l’application vers le bureau.

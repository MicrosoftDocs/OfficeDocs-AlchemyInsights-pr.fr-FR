# <a name="get-started-with-microsoft-edge--bing"></a>Mise en place de Microsoft Edge + Bing

Aucune configuration supplémentaire n’est nécessaire pour commencer avec Microsoft Edge + Bing. Pour rechercher des informations relatives au travail, il vous suffit d’entrer votre requête dans la barre d’adresses de Microsoft Edge.

Pour plus d’informations, voir [Obtenir plus d’informations avec Microsoft Edge + Bing](https://go.microsoft.com/fwlink/?linkid=2152963).

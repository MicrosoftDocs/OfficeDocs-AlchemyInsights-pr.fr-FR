# <a name="public-folder-access-from-mobile-devices"></a>Accès aux dossiers publics à partir d’appareils mobiles

L’accès aux dossiers publics ne fonctionne pas avec les appareils mobiles, utilisez OWA (Outlook sur le Web) ou Outlook pour les clients de bureau Windows ou Outlook pour Mac pour accéder aux dossiers publics.

Si vous souhaitez disposer de fonctionnalités similaires aux dossiers publics et que le contenu soit accessible sur les appareils mobiles, consultez la [rubrique en savoir plus sur les groupes Microsoft 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) pour une meilleure alternative.

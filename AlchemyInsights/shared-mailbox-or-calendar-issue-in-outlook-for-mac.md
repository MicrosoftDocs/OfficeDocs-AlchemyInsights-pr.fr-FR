# <a name="shared-mailbox-or-calendar-issue-in-outlook-for-mac"></a>Problème de boîte aux lettres ou le calendrier partagé dans Outlook pour Mac ?

Nous pouvons vous aider plus rapide si nous contacter directement à partir de l’application. **Allez pour Mac de l’utilisateur concerné, ouvrez Outlook, cliquez sur Aide \> contacter le support technique**. 
  
Pour plus d’informations sur l’utilisation d’Outlook pour la prise en charge de Mac dans l’application, consultez cet article : [contacter le Support nouvelle fonctionnalité 2016 Outlook pour Mac](https://answers.microsoft.com/msoffice/forum/msoffice_outlook-mso_mac-mso_mac2016/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517.aspx)

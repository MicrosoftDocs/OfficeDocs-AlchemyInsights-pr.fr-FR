# <a name="change-the-domain-for-office-365-group"></a>Modifier le domaine pour le groupe Office 365

Si vous souhaitez modifier le domaine pour un groupe Office 365 lors de la création, suivez les étapes de la procédure [choisir le domaine à utiliser lors de la création de groupes office 365](https://support.office.com/article/7cf5655d-e523-4bc3-a93b-3ccebf44a01a.aspx).
  
Une fois que le groupe Office 365 est déjà créé, il n’existe actuellement aucun moyen de modifier le domaine. La seule façon de créer un groupe Office 365 et de supprimer l’ancien si nécessaire (des adresses de messagerie supplémentaires peuvent être ajoutées via le [Centre d’administration Exchange](https://outlook.office365.com/ecp.aspx)).

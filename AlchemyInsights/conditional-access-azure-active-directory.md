# <a name="conditional-access"></a>Accès conditionnel

<p>L’accès conditionnel est une fonctionnalité d’Azure Active Directory. Avec l’accès conditionnel, vous pouvez implémenter des décisions de contrôle d’accès automatisées pour accéder à vos applications Cloud basées sur des conditions. Pour plus d’informations, reportez-vous à la <a href="https://docs.microsoft.com/en-us/azure/active-directory/conditional-access/overview">Configuration des stratégies basées sur l’emplacement</a>.</p> <p>En tant qu’administrateur informatique, vous pouvez également contrôler l’accès aux ressources SharePoint et OneDrive en fonction des emplacements réseau définis ou des appareils que vous approuvez. Pour plus d’informations, consultez les liens ci-dessous.</p> <ul> <li><a href="https://docs.microsoft.com/en-us/sharepoint/control-access-based-on-network-location">Contrôler l’accès aux données SharePoint Online et OneDrive en fonction de l’emplacement réseau</a></li> <li><a href="https://docs.microsoft.com/en-us/sharepoint/control-access-from-unmanaged-devices">Contrôler l’accès depuis des appareils enregistrés</a></li> </ul>

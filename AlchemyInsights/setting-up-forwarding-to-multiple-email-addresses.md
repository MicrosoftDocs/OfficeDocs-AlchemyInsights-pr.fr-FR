# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>Configuration du transfert vers plusieurs adresses de messagerie

Si vous devez configurer le transfert du courrier vers un groupe d’utilisateurs, créez une liste de distribution, ajoutez-y plusieurs utilisateurs, puis configurez le transfert vers cette liste. Tout d’abord, dans le centre d’administration, **accédez à groupes groupes**  >  [Groups](https://portal.office.com/adminportal/home#/groups).
  
1. Créer un groupe.

2. Ajoutez plusieurs utilisateurs.

3. Configurez le transfert des boîtes aux lettres vers ce groupe.

# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>Configuration du transfert vers plusieurs adresses de messagerie

Si vous devez configurer le transfert du courrier vers un groupe d’utilisateurs, créez une liste de distribution, ajoutez-y plusieurs utilisateurs, puis configurez le transfert sur cette liste. Tout d’abord, accédez à [gérer les groupes](https://portal.office.com/adminportal/home#/groups)
  
1. Créer une *liste de distribution*

2. Ajouter plusieurs utilisateurs

3. Configurez ensuite le transfert des boîtes aux lettres vers ce groupe.

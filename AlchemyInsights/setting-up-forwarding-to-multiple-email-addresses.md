# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>Configuration d’un transfert vers plusieurs adresses de messagerie

Tout d’abord, accédez à [Gérer les groupes](https://portal.office.com/adminportal/home#/groups)
  
1. Créer une *liste de distribution* 
    
2. Ajouter plusieurs utilisateurs
    
3. Puis configurer le transfert de boîte aux lettres à ce groupe

# <a name="create-a-sensitivity-label"></a>Créer une étiquette de confidentialité

Pour créer une étiquette de confidentialité, effectuez la procédure suivante :

1. Accédez au **[Centre de sécurité et conformité Office 365](https://sip.protection.office.com/) > Classification > Étiquettes de sensibilité**.
2. Cliquez sur **+ Créer une étiquette** pour démarrer l’assistant **Nouvelle étiquette de confidentialité**. Pour plus d’informations sur les paramètres de chiffrement et l’étiquetage automatique, consultez les rubriques [Restreindre l’accès au contenu à l’aide du chiffrement dans les étiquettes de niveau de confidentialité](https://docs.microsoft.com/microsoft-365/compliance/encryption-sensitivity-labels) et [Appliquer automatiquement une étiquette sensibilité au contenu](https://docs.microsoft.com/microsoft-365/compliance/apply-sensitivity-label-automatically), respectivement.
3. Répétez ces étapes pour créer d’autres étiquettes. Toutefois, si vous voulez créer une sous-étiquette, commencez par sélectionner l’étiquette parent et cliquez sur **...** pour **Plus d’actions**, puis **Ajouter une sous-étiquette**.
4. Lorsque vous avez créé les étiquettes dont vous avez besoin, vérifiez leur ordre et déplacez-les vers le haut ou vers le bas si nécessaire. Pour modifier l’ordre d’une étiquette, cliquez sur **...** pour **Plus d’actions**, puis cliquez sur **Déplacer vers le haut** ou **Déplacer vers le bas**.

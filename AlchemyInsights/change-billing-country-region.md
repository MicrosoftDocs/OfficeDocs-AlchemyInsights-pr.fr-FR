# <a name="change-billing-country-or-region"></a>Modifier le pays ou la région de facturation

Il n’est pas possible de modifier le pays ou la région à partir des comptes de facturation dans le centre  >   d’administration Microsoft 365 de facturation. Vous devez vous inscrire à un nouveau compte et choisir le pays/la région souhaité(s).

# <a name="cant-read-email-in-google-chrome-with-adblock"></a>Impossible de lire le courrier dans Google Chrome avec AdBlock

Certains utilisateurs sont rencontre des problèmes lors de la lecture du courrier électronique ou composer un nouveau message lors de l’utilisation de Google Chrome avec l’extension AdBlock. Essayez les solutions suivantes :
  
- Utiliser [Microsoft Edge](https://go.microsoft.com/fwlink/p/?linkid=2001503&amp;clcid=0x409) ou un autre navigateur web. 
    
- Exclure Outlook.com les paramètres AdBlock. Dans le coin supérieur droit de Google Chrome, sélectionnez **AdBlock** , puis sélectionnez **ne pas s’exécuter sur les pages de ce domaine**. Actualisez votre navigateur pour charger Outlook.com à nouveau.

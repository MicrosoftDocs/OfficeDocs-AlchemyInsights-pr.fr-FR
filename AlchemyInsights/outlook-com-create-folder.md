# <a name="create-a-folder"></a>Créer un dossier

1. En bas du volet des dossiers, sélectionnez **nouveau dossier**. Vous devrez peut-être faire défiler le volet des dossiers vers le bas pour afficher le bouton **nouveau dossier** .
2. Dans la zone de texte, entrez un nom pour le dossier, puis appuyez sur entrée.

Pour en savoir plus, consultez [la rubrique créer un dossier dans Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).

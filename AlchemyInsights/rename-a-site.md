# <a name="rename-a-site"></a>Renommer un site

Pour modifier le nom d’un site, par exemple, à partir de https://contoso.sharepoint.com/sites/sample1 à https://contoso.sharepoint.com/sites/sample2, vous avez besoin créer un nouveau site, puis déplacez tout le contenu de l’ancien site vers le nouveau. Pour obtenir des informations sur le déplacement d’éléments dans les bibliothèques de documents, voir [déplacer ou copier des éléments à partir d’une bibliothèque de documents SharePoint](https://go.microsoft.com/fwlink/?Linkid=2018691).
  
Remarque : Si vous essayez de modifier le nom de domaine (autrement dit, la partie contoso de l’URL ci-dessus) dans une URL en ligne de SharePoint, qui n’est pas possible. Consultez [modifier vos domaines](https://go.microsoft.com/fwlink/?Linkid=2018696).

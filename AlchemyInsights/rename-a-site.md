# <a name="rename-a-sharepoint-site"></a>Renommer un site SharePoint

Suivez les étapes décrites dans [cet article](https://docs.microsoft.com/sharepoint/change-site-address) pour modifier le nom d’un site, par exemple, https://contoso.sharepoint.com/sites/sample1 de https://contoso.sharepoint.com/sites/sample2à.

**Remarque :** Vous ne pouvez pas modifier le nom de domaine (dans l’exemple ci-dessus, la partie contoso de l’URL) dans une URL SharePoint Online. 

Pour plus d’informations, voir :

- [Modifier votre nom de domaine SharePoint](https://go.microsoft.com/fwlink/?Linkid=2018696)
- [Erreurs lorsque vous renommez une adresse de site SharePoint](https://support.office.com/article/errors-when-you-rename-a-sharepoint-site-address-165b7c11-1325-4813-b160-ecbe87bc1a86)

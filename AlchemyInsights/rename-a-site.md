# <a name="rename-a-site"></a>Renommer un site

Pour modifier le nom d’un site, par exemple, pour passer de https://contoso.sharepoint.com/sites/sample2 à [](https://go.microsoft.com/fwlink/?Linkid=2018691), vous devez créer un site, puis déplacer le contenu de l’ancien site vers le nouveau. Pour plus d’informations sur le déplacement des éléments dans les bibliothèques de documents, reportez-vous à l’article Déplacer ou copier des éléments à partir d’une bibliothèque de documents SharePoint.
  
Remarque : Si vous essayez de modifier le nom de domaine (autrement dit, la partie contoso de l’URL ci-dessus) dans une URL en ligne de SharePoint, qui n’est pas possible. Consultez [modifier vos domaines](https://go.microsoft.com/fwlink/?Linkid=2018696).

# <a name="rename-a-sharepoint-site"></a>Renommer un site SharePoint

Pour modifier le nom d’un site, par exemple, de https://contoso.sharepoint.com/sites/sample1 à https://contoso.sharepoint.com/sites/sample2, vous devez créer un nouveau site, puis déplacer tout le contenu de l’ancien site vers le nouveau. Pour plus d’informations sur le déplacement d’éléments dans les bibliothèques de documents, voir [déplacer ou copier des éléments d’une bibliothèque de documents SharePoint](https://go.microsoft.com/fwlink/?Linkid=2018691).
  
Remarque: Si vous essayez de modifier le nom de domaine (autrement dit, la partie contoso de l’URL ci-dessus) dans une URL SharePoint Online, cela n’est pas possible. Consultez [la rubrique modifier vos domaines](https://go.microsoft.com/fwlink/?Linkid=2018696).

# <a name="enable-multi-factor-authentication"></a>Activer l’authentification multifacteur

1. Accédez aux **utilisateurs** \> **utilisateurs actifs**
    
2. Cliquez sur **plus de** \> **authentification multifacteur de Azure le programme d’installation**. 
    
3. Recherchez l’ou les utilisateurs auxquels vous souhaitez activer pour MFA. Afin de voir tous les utilisateurs, vous devrez peut-être modifier l’affichage de l’état d’authentification multifacteur en haut.
    
4. Case à cocher en regard des utilisateurs que vous souhaitez activer.
    
5.  Dans le volet d’informations utilisateur, sous **étapes rapides** vous voyez activer et gérer les paramètres utilisateur. Cliquez sur **Activer**. 
    
6. Dans la boîte de dialogue qui s’ouvre, **cliquez sur Activer l’authentification multifacteur**.

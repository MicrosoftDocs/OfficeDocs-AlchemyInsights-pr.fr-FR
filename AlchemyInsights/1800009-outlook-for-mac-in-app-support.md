# <a name="in-app-support-in-outlook-for-mac"></a>Prise en charge dans l’application dans Outlook pour Mac

Pour recevoir la prise en charge dans l’application pour Outlook pour les problèmes Mac, ouvrez Outlook et cliquez sur Aide \> contacter le Support. Tapez votre question et un membre d’Outlook pour Mac d’être contacté par équipe de support technique. [Contactez le support technique dans Outlook pour Mac](https://go.microsoft.com/fwlink/?linkid=2002400&amp;clcid=0x409)

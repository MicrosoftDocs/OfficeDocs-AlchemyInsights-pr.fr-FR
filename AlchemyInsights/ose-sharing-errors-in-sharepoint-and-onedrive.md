# <a name="ose-sharing-errors"></a>Erreurs de partage OSE

Cet article décrit les différentes erreurs que vous pouvez rencontrer lors du partage de fichiers ou de dossiers à partir de SharePoint Online ou de OneDrive entreprise dans Microsoft 365. Vous devez être un administrateur SharePoint général de votre organisation pour corriger ces erreurs. Si vous n’êtes pas administrateur, contactez le support technique en indiquant le code d’erreur.

Pour plus d’informations sur les codes d’erreur OSE, consultez la rubrique [partage des erreurs dans SharePoint et OneDrive](https://docs.microsoft.com/sharepoint/sharepoint-onedrive-error-message).

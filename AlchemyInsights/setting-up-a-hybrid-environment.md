# <a name="setting-up-a-hybrid-environment"></a>Configuration d'un environnement hybride

Pour permettre à vos utilisateurs de travailler en toute transparence, quel que soit le stockage des données sur des serveurs locaux ou dans Office 365, vous devrez peut-être configurer un environnement hybride. Pour obtenir une vue d'ensemble de vos options, consultez l' [intégration d'Office 365 dans les environnements locaux](https://support.office.com/article/263faf8d-aa21-428b-aed3-2021837a4b65.aspx).

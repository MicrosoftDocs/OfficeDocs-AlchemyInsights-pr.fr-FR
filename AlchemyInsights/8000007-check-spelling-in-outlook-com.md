# <a name="use-your-browser-to-check-spelling"></a>Utilisez votre navigateur pour vérifier l’orthographe

Il n’existe aucune option de vérification orthographique dans Outlook.com. Pour vérifier l’orthographe, vous devez utiliser votre navigateur web. Vérification de l’orthographe est disponible dans Microsoft Edge, Internet Explorer 10 et versions ultérieures et les versions actuelles de Firefox, Chrome et Safari. Vérifiez les options de votre navigateur web pour en savoir plus sur la façon de vérifier l’orthographe.

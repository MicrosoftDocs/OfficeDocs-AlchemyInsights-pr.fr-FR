# <a name="enable-or-disable-forms-pro"></a>Activer ou désactiver Forms Pro

Si vous essayez d’activer Microsoft Forms Pro, elle est incluse dans le cadre de la licence [Dynamics 365 Enterprise](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant) . En guise d’alternative, un administrateur peut acheter une [licence Microsoft Forms Pro](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant). Vous pouvez également [supprimer une licence Microsoft Forms Pro](https://docs.microsoft.com/forms-pro/purchase#disable-microsoft-forms-pro-for-a-user-1) d’un utilisateur.

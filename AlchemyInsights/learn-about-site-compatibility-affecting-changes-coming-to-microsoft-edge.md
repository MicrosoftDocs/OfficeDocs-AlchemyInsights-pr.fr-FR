# <a name="learn-about-site-compatibility-affecting-changes-coming-to-microsoft-edge"></a>En savoir plus sur les modifications qui affectent la compatibilité des sites dans Microsoft Edge

Le web évolue constamment pour améliorer l’expérience utilisateur, la sécurité et la confidentialité. Certaines modifications sont suffisamment importantes pour affecter les fonctionnalités des pages existantes. L’équipe Microsoft Edge met à jour les pages à mesure que la réflexion évolue, que les chronologies s’unifient et que des [modifications sont annoncées.](https://go.microsoft.com/fwlink/?linkid=2135534)

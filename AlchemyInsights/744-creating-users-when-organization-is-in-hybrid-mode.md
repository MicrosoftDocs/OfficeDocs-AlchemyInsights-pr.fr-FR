# <a name="create-users-in-hybrid-deployments"></a>Créer des utilisateurs dans les déploiements hybrides

Lorsque votre organisation est un déploiement hybride entre un environnement local et d’Office 365, vous devez créer tous les objets utilisateur dans Active Directory local. Vous créez les objets utilisateur dans l’unité d’organisation (UO) appropriée afin qu’ils sont synchronisées dans votre organisation Office 365.

# <a name="screen-sharing-not-working-in-teams"></a>Le partage d’écran ne fonctionne pas dans Teams

Le partage d’écran ne fonctionne pas dans les situations suivantes :

- L’utilisateur est sur Teams sur le web. L’utilisateur doit basculer vers la version installée de Teams.
- Le partage d’écran est désactivé sur un appareil Mac. Pour activer le partage d’écran, voir [Partager du contenu sur un Mac](https://support.microsoft.com/office/share-content-in-a-meeting-in-teams-fcc2bf59-aecd-4481-8f99-ce55dd836ce8#bkmk_sharecontentonmac).
- L’administrateur Teams a bloqué le partage d’écran. Consultez votre administrateur Teams.  
    
Pour plus d’informations, voir [Partager du contenu dans une réunion Teams](https://support.microsoft.com/office/share-content-in-a-meeting-in-teams-fcc2bf59-aecd-4481-8f99-ce55dd836ce8).

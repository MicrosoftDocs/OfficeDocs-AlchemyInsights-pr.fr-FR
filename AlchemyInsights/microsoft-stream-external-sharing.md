# <a name="microsoft-stream-external-sharing"></a>Partage externe Microsoft Stream

Microsoft Stream ne prend pas encore en charge les scénarios de partage externe pour les invités ou les personnes externes à votre organisation. Nous connaissons que ces fonctionnalités sont hautement nécessaires, et nous travaillons sur les plans de création de ces fonctionnalités.

- [Partage externe dans le flux](https://docs.microsoft.com/stream/portal-share-video#external-sharing)

- [Partager vos vidéos dans Microsoft Stream](https://docs.microsoft.com/stream/portal-share-video)

# <a name="how-to-change-your-layout"></a>Comment modifier la disposition

Il est facile de modifier les paramètres dans Outlook.com de sorte qu'il se présente et se comporte comme vous le souhaitez. Sélectionnez **paramètres** dans la barre d'outils pour afficher vos choix.

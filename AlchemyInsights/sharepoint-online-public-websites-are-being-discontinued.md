# <a name="sharepoint-online-public-websites-are-being-discontinued"></a>Les sites Web publics SharePoint Online sont imminente

Si vous avez créé un site Web public SharePoint Online, vous pouvez continuer à l’utiliser jusqu’au 31 mars, 2018. Si vous souhaitez continuer à avoir un site Web public, vous devez créer un nouveau avec un nouveau fournisseur. Le contenu de votre site Web public SharePoint Online sera supprimé, vous devez copier tout contenu que vous souhaitez conserver.
  
Une bannière apparaît maintenant dans la partie supérieure de votre site Web public SharePoint Online en tant que notification que le site sera supprimé. Vous ne pouvez pas désactiver cette bannière. Pour obtenir des informations sur l’enregistrement et la copie du contenu d’un site Web partenaire, voir [plus d’informations sur le processus d’arrêt de fonctionnalité](https://go.microsoft.com/fwlink/?linkid=866980).

# <a name="manage-search-dictionaries"></a>Gérer les dictionnaires de recherche

Dictionnaires de recherche dans SharePoint online sont gérés dans le magasin de termes de SharePoint Online qui se trouve dans le centre d’administration en ligne SharePoint classique. Pour plus d’informations sur la façon d’ajouter ou de modifier des dictionnaires de recherche, voir [Gérer les dictionnaires de recherche](https://go.microsoft.com/fwlink/?linkid=2044669&amp;clcid=0x409).
  
Si vous ne voyez pas un terme de dictionnaire de recherche défini apparaissent dans le magasin de termes, attendez 24 heures. Si le dictionnaire toujours ne pas affichées après 24 heures, veuillez vous connecter un ticket de support.

# <a name="how-do-i-search-for-an-email"></a>Comment rechercher un courrier électronique ?

Vous pouvez rechercher un message électronique en consultant le contact qui vous a envoyé le courrier électronique, ou en recherchant un mot clé contenu dans le courrier électronique.

Appuyez sur la loupe dans la barre de navigation inférieure. Tapez ensuite votre demande de recherche dans la zone située en haut de votre écran. 

**Conseil :** Vous pouvez double-cliquer sur la loupe pour activer la zone de recherche. Vos résultats apparaîtront lorsque vous taperez votre requête.

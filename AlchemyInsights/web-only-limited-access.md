# <a name="web-only-limited-access"></a>Web-Only accès limité

Web-Only accès limité est une variante du niveau d’autorisation « accès limité » qui permet uniquement aux utilisateurs d’accéder à l’objet Web.

Pour plus d’informations, consultez la rubrique [niveaux d’autorisation par défaut](https://docs.microsoft.com/sharepoint/understanding-permission-levels#default-permission-levels).

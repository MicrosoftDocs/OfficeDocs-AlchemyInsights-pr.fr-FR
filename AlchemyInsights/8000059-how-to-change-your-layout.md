# <a name="how-to-change-your-layout"></a>Comment faire pour modifier votre mise en page

Il est facile de modifier les paramètres dans Outlook.com pour qu’il se présente et comporte comme vous le souhaitez. Sélectionnez **paramètres de** la barre d’outils pour afficher votre choix.

# <a name="new-office-365-e1-trial"></a>Nouvelle version d'essai d'Office 365 E1

À compter du 1er juillet 2020, la licence d'essai Office 365 E1 n'est plus disponible. Si vous avez besoin de licencer des utilisateurs pour Microsoft Teams, lisez la description du [service Microsoft Teams](https://docs.microsoft.com/office365/servicedescriptions/teams-service-description) pour obtenir la liste des abonnements payants qui incluent Teams. Les organisations éligibles peuvent également utiliser la version gratuite de [Teams](https://support.office.com/article/Welcome-to-Microsoft-Teams-free-6d79a648-6913-4696-9237-ed13de64ae3c)ou les employés peuvent activer l'expérience [exploratoire Teams.](https://docs.microsoft.com/MicrosoftTeams/teams-exploratory)

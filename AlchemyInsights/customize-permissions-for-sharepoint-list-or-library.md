# <a name="how-to-customize-permissions-for-a-sharepoint-list-or-library"></a>Procédure de personnalisation des autorisations pour une liste ou une bibliothèque SharePoint

En tant qu’administrateur ou propriétaire d’une bibliothèque, d’une liste ou d’une enquête, vous pouvez modifier les autorisations pour permettre à un groupe de personnes d’accéder aux données dont elles ont besoin tout en limitant les autres.

[Personnaliser les autorisations pour une liste ou une bibliothèque SharePoint](https://support.office.com/article/customize-permissions-for-a-sharepoint-list-or-library-02d770f3-59eb-4910-a608-5f84cc297782)

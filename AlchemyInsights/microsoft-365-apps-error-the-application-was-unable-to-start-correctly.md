# <a name="microsoft-365-apps-error-the-application-was-unable-to-start-correctly-0xc0000142"></a>Erreur applications Microsoft 365 : l’application n’a pas pu démarrer correctement (0xc0000142)

Pour corriger cette erreur, essayez les options suivantes :

- Effectuez une réparation en ligne d’Office et redémarrez l’ordinateur. Pour plus d’informations, consultez [Réparer une application Office](https://support.microsoft.com/office/repair-an-office-application-7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b).
- Accédez à **Démarrer**  >  **Exécuter**, puis tapez **services.msc**. Cliquez avec le bouton droit sur  **Service Microsoft Office « démarrer en un clic »**, puis sélectionnez **Redémarrer**.
- Réparez les fichiers système manquants ou endommagés à l’aide du vérificateur des fichiers système, puis redémarrez l’ordinateur. Pour plus d’informations, consultez [Utiliser l’outil Vérificateur des fichiers système pour réparer les fichiers système manquants ou endommagés](https://support.microsoft.com/help/929833/use-the-system-file-checker-tool-to-repair-missing-or-corrupted-system).
- Désinstaller et réinstaller Office. Pour plus d’information, consultez [Désinstaller Office d’un PC](https://support.microsoft.com/office/uninstall-office-from-a-pc-9dd49b83-264a-477a-8fcc-2fdf5dbf61d8).

Pour plus d’informations, voir :  

- [Erreur lors de l’ouverture des applications Microsoft Office](https://support.office.com/article/error-when-opening-microsoft-office-apps-b84b6a63-4b8c-46ec-ae9a-ad91d6160d72)  

- [Je rencontre une erreur du type « a cessé de fonctionner » quand je démarre des applications Office sur mon PC](https://support.office.com/article/i-get-a-stopped-working-error-when-i-start-office-applications-on-my-pc-52bd7985-4e99-4a35-84c8-2d9b8301a2fa)

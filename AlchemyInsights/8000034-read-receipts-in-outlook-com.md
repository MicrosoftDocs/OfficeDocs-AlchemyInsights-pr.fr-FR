# <a name="read-receipts-in-outlookcom"></a>Confirmations de lecture dans Outlook.com

Vous ne pouvez pas demander une confirmation de lecture dans Outlook.com, mais vous pouvez demander des confirmations de lecture pour les messages Outlook.com envoyés à l’aide d’Outlook pour Windows. Pour savoir comment procéder, voir [Ajouter et demande de confirmations de lecture et les notifications de remise](https://go.microsoft.com/fwlink/p/?linkid=874355).
  
Pour choisir comment Outlook.com répond aux demandes de confirmation de lecture, accédez à **paramètres**![paramètres](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png) \> **Options** \> **messagerie** \> **le traitement automatique** \> **confirmations de lecture**. 
  
Dans la version bêta Outlook.com, sélectionnez **paramètres**![paramètres](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png) et recherchez **les confirmations de lecture**.

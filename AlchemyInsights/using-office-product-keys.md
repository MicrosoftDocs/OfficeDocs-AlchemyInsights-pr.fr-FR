# <a name="using-office-product-keys"></a>Utilisation des clés de produit Office

**Clés de produit Office**

- Si votre achat d’Office est fourni avec une clé de produit, consultez [la rubrique où entrer votre clé de produit Office](https://support.office.com/article/Where-to-enter-your-Office-product-key-0a82e5ae-739e-4b92-a6f4-2ec780c185db).

- Pour obtenir de l’aide sur une clé de produit Office 365 famille ou personnel, consultez la rubrique [utilisation de clés de produit avec Office](https://support.office.com/article/using-product-keys-with-office-12a5763a-d45c-4685-8c95-a44500213759).

**Activation de l’abonnement Office** 
- Si vous avez acheté un abonnement Microsoft 365 auprès d’un détaillant, reportez-vous à [la rubrique Enter your product key for microsoft 365 Apps for Business](https://docs.microsoft.com/microsoft-365/commerce/enter-your-product-key).

- Si un message d’erreur s’affiche lorsque vous tentez d’échanger une clé de produit, consultez [la rubrique problèmes liés à votre clé de produit Microsoft 365 pour les entreprises ?](https://docs.microsoft.com/microsoft-365/commerce/product-key-errors-and-solutions)

# <a name="set-up-microsoft-information-protection"></a>Configurer la protection des données Microsoft

Bénéficiez d’une vue d’ensemble des fonctionnalités que vous pouvez appliquer à votre stratégie de protection des informations pour garantir la protection de vos informations sensibles. Utilisez une approche de cycle de vie à quatre étapes consistant à découvrir, à classer, à protéger, puis à contrôler les informations sensibles. Le guide « Configuration de la protection des données Microsoft » fournit des conseils sur chacune de ces étapes.

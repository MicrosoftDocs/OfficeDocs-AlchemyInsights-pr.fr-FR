# <a name="get-in-app-support-for-the-outlook-mobile-app"></a>Obtenir la prise en charge dans l’application pour l’application mobile Outlook

1. Dans l’application mobile Outlook, ouvrez le **Menu**![bouton du Menu](media/265b9089-9630-42dd-a244-d9a412d8fe47.png).
    
2. Sélectionnez le point d’interrogation![bouton aide le](media/3b8cbf5a-6ced-4d79-b53c-fa82045c3e25.png) pour ouvrir le **aide &amp; commentaires** écran. 
    
3. Appuyez sur **Contacter le support technique** et expliquent votre problème. 
    
4. Appuyez sur **Envoyer** et que vous recevez une notification dans l’application lorsque vous en contact avec un agent de prise en charge. 
    
Pour obtenir des instructions détaillées, consultez la rubrique [obtenir de l’aide dans l’application pour Outlook pour iOS et Android](https://support.office.com/article/218a22d1-9fa5-4889-b689-de1c63493243.aspx#ID0EAABAAA=Contact_Support).

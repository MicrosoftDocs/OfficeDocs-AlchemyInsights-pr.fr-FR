# <a name="workflows-in-sharepoint"></a>Flux de travail dans SharePoint

<p>Si les flux de travail SharePoint n’envoient pas d’e-mails, il se peut que votre organisation ait rencontré des limites d’expéditeurs Exchange Online.&nbsp;</p> <p>Le message d’erreur «le flux de travail est suspendu» peut se produire si vous avez l’un des éléments suivants:</p> <ul> <li>Vous disposez d’un flux de travail dans SharePoint Online qui utilise le type de plateforme de flux de travail SharePoint 2010 ou SharePoint 2013.</li> <li>Le flux de travail est configuré pour envoyer un message électronique personnalisé à plus de 200 utilisateurs à la fois, plus de 10 000 destinataires par jour ou plus de 30 messages par minute.</li> <li>Lorsque vous exécutez le flux de travail, le message électronique n’est pas envoyé et vous remarquez le message d’erreur, le <strong>statut Internal est défini sur suspendu</strong> ou <strong>Impossible d’envoyer à un destinataire</strong> est affiché.</li> </ul> <p>Pour plus d’informations, reportez-vous à l' <a href="https://support.office.com/en-us/article/-daily-email-limit-has-exceeded-and-your-workflow-has-been-suspended-or-unable-to-send-to-a-recipient-error-in-a-sharepoint-online-workflow-89d02169-5fa6-4259-affc-73edb6ca9fb6?ui=en-US&amp;rs=en-US&amp;ad=US">article</a>suivant.</p>
